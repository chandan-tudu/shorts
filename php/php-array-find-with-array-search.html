PHP Array Find With array_search()

In PHP, you can use the array_search() function to find the key associated with a particular value in an array.

If you want to find the value itself, you can simply access it using the key obtained from array_search().

Here's an example:

<?php
$fruits = array("apple", "banana", "orange", "grape");

// Find the key of "orange" in the array
$key = array_search("orange", $fruits);

if ($key !== false) {
    echo "Key of 'orange': $key<br>";

    // Access the value using the key
    $value = $fruits[$key];
    echo "Value at key $key: $value";
} else {
    echo "'orange' not found in the array";
}
?>

In this example, array_search("orange", $fruits) returns the key associated with the value "orange" in the $fruits array. If the value is found, it returns the corresponding key; otherwise, it returns false.

Keep in mind that array_search() performs a loose comparison, so if you need to perform a strict comparison, you may want to use a loop or other methods, depending on your specific requirements.

If you're working with PHP 7 or later and want to find a value and its associated key using a callback function, you can use the array_filter() function with a custom callback:

<?php
$fruits = array("apple", "banana", "orange", "grape");

// Find the value and key using a callback
$result = array_filter($fruits, function ($value) {
    return $value === "orange";
});

if (!empty($result)) {
    $key = key($result);
    echo "Key of 'orange': $key<br>";
    echo "Value at key $key: $result[$key]";
} else {
    echo "'orange' not found in the array";
}
?>

This example uses array_filter() with a callback to find the value "orange" in the $fruits array and obtain both the key and the value.
