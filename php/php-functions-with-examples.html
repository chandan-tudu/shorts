PHP Functions With Examples

In PHP, a function is a block of reusable code that performs a specific task. Functions help organize code into manageable and modular pieces, making it easier to understand, maintain, and reuse. Here is the basic syntax for defining and using a function in PHP:

1. Defining a Function:

function functionName($parameter1, $parameter2, ...) {
    // Code to be executed
    return $result; // Optional return statement
}
  • functionName: The name of the function.
  • $parameter1, $parameter2, ...: Parameters (input) that the function can accept.
  • {}: The code block to be executed.
  • return: Optional statement to return a value from the function.

Example:

function greet($name) {
    echo "Hello, $name!";
}

// Calling the function
greet("John");
// Outputs: Hello, John!

2. Function with Return Value:

function add($a, $b) {
    $sum = $a + $b;
    return $sum;
}

$result = add(5, 3);
echo $result;
// Outputs: 8

3. Default Parameter Values:

function multiply($a, $b = 2) {
    $product = $a * $b;
    return $product;
}

$result1 = multiply(5);
echo $result1;
// Outputs: 10

$result2 = multiply(5, 3);
echo $result2;
// Outputs: 15

4. Variable Scope:

Variables inside a function have local scope, meaning they are only accessible within that function. You can also use global variables within a function, but you need to declare them as global.

5. Anonymous Functions (Closures):

Anonymous functions, also known as closures, allow you to create functions on the fly without specifying a name.

$add = function($a, $b) {
    return $a + $b;
};

$result = $add(5, 3);
echo $result;
// Outputs: 8

These are the basics of PHP functions. They play a crucial role in writing modular and reusable code. Functions can be used for a wide range of tasks, from simple calculations to complex operations.
