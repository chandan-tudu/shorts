What Are Constants In PHP With Examples

In PHP, constants are like variables, but once they are defined, they cannot be changed or redefined during the execution of the script.

Constants are useful for storing values that remain constant throughout the execution of a script, such as configuration settings or fixed values.

Here's how you can define and use constants in PHP:

1. Defining Constants:

You can use the define() function to define a constant. Constants have a case-sensitive name and follow the same rules as variable names in PHP.

define("CONSTANT_NAME", "constant_value");

Example:

define("PI", 3.14);
define("GREETING", "Hello, World!");

2. Using Constants:

Once defined, you can use constants in your script as you would use variables.

echo PI;        // Outputs: 3.14
echo GREETING;  // Outputs: Hello, World!

Constants can be used anywhere in your script, including inside functions, classes, and loops.

3. Case-Sensitivity:

Constants in PHP are case-sensitive by default. For example:

define("MyConstant", "Some value");
echo MyConstant;   // Outputs: Some value
echo myconstant;   // This would result in an error, as constant names are case-sensitive

4. Checking if a Constant is Defined:

You can use the defined() function to check if a constant is defined before using it:

if (defined("PI")) {
    echo PI;  // Outputs: 3.14
} else {
    echo "PI is not defined";
}

5. Magic Constants:

PHP also provides a set of special constants called "magic constants" that change based on their usage. Examples include __LINE__ (current line number), __FILE__ (full path and filename of the script), and __FUNCTION__ (function name).

echo __LINE__;      // Outputs: Current line number
echo __FILE__;      // Outputs: Full path and filename of the script
echo __FUNCTION__;  // Outputs: Function name

Constants are a useful way to make your code more readable, maintainable, and less error-prone by ensuring that certain values remain constant throughout your script's execution.
