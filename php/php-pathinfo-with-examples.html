PHP pathinfo() With Examples

In PHP, the pathinfo() function is used to parse information about a file path. It returns an associative array containing information about the path, such as the directory name, basename, extension, and filename.

Here's an example:

<?php
// File path
$filePath = '/path/to/example.txt';

// Get information about the file path
$pathInfo = pathinfo($filePath);

// Display the path information
print_r($pathInfo);
?>

In this example, the pathinfo() function is used to extract information from the file path /path/to/example.txt. The resulting array will look something like this:

Array
(
    [dirname] => /path/to
    [basename] => example.txt
    [extension] => txt
    [filename] => example
)
  • dirname: The directory name of the path.
  • basename: The file name with extension.
  • extension: The file extension (if any).
  • filename: The file name without extension.

You can access each element of the array using the corresponding key. For example:

<?php
echo 'Directory: ' . $pathInfo['dirname'] . '<br>';
echo 'Basename: ' . $pathInfo['basename'] . '<br>';
echo 'Extension: ' . $pathInfo['extension'] . '<br>';
echo 'Filename: ' . $pathInfo['filename'] . '<br>';
?>

This will output:

Directory: /path/to
Basename: example.txt
Extension: txt
Filename: example

You can use pathinfo() to easily retrieve and manipulate information about file paths in your PHP applications.
