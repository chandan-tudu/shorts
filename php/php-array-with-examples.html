PHP Array With Examples

In PHP, an array is a versatile data structure that allows you to store multiple values in a single variable. Each value in an array is associated with a unique key or index. Here are some examples of using arrays in PHP:

1. Numeric Arrays:

Numeric arrays use integers as keys. The index starts from 0.

// Creating a numeric array
$colors = array("Red", "Green", "Blue");

// Accessing elements
echo $colors[0]; // Outputs: Red
echo $colors[1]; // Outputs: Green
echo $colors[2]; // Outputs: Blue

// Modifying elements
$colors[1] = "Yellow";
echo $colors[1]; // Outputs: Yellow

// Adding elements
$colors[] = "Purple";
echo $colors[3]; // Outputs: Purple

// Counting elements
$count = count($colors);
echo $count; // Outputs: 4

2. Associative Arrays:

Associative arrays use named keys.

// Creating an associative array
$person = array(
    "name" => "John",
    "age" => 30,
    "city" => "New York"
);

// Accessing elements
echo $person["name"]; // Outputs: John
echo $person["age"];  // Outputs: 30
echo $person["city"]; // Outputs: New York

// Modifying elements
$person["age"] = 31;
echo $person["age"];  // Outputs: 31

// Adding elements
$person["gender"] = "Male";
echo $person["gender"]; // Outputs: Male

// Counting elements
$count = count($person);
echo $count; // Outputs: 4

3. Multidimensional Arrays:

Multidimensional arrays are arrays within arrays.

// Creating a multidimensional array
$students = array(
    array("John", 25, "Male"),
    array("Alice", 22, "Female"),
    array("Bob", 28, "Male")
);

// Accessing elements
echo $students[0][0]; // Outputs: John
echo $students[1][1]; // Outputs: 22
echo $students[2][2]; // Outputs: Male

4. Looping Through Arrays:

// Using foreach to loop through a numeric array
$fruits = array("Apple", "Banana", "Orange");
foreach ($fruits as $fruit) {
    echo $fruit . " ";
}
// Outputs: Apple Banana Orange

// Using foreach to loop through an associative array
$person = array("name" => "John", "age" => 30, "city" => "New York");
foreach ($person as $key => $value) {
    echo "$key: $value, ";
}
// Outputs: name: John, age: 30, city: New York,

Arrays in PHP are dynamic, flexible, and widely used in various programming scenarios. They are a fundamental part of PHP programming, and understanding how to work with them is essential.
