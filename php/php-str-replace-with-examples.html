PHP str_replace() With Examples

In PHP, the str_replace() function is used to replace occurrences of a specified substring with another substring in a given string. The basic syntax of str_replace() is as follows:

str_replace(search, replace, subject, count);
  • search: The substring you want to replace.
  • replace: The substring to replace the search string with.
  • subject: The original string or an array of strings where replacements will be performed.
  • count (optional): A variable to store the number of replacements performed.

Here's an example:

<?php
// Original string
$originalString = "Hello, world! Hello, universe!";

// Replace "Hello" with "Hi"
$newString = str_replace("Hello", "Hi", $originalString);

// Display the new string
echo $newString;
?>

In this example, the str_replace() function is used to replace every occurrence of "Hello" with "Hi" in the $originalString. The output will be:

Hi, world! Hi, universe!

If you want to replace multiple substrings at once, you can pass arrays as the search and replace parameters. For example:

<?php
// Original string
$originalString = "apple orange banana";

// Replace multiple fruits
$search = array("apple", "orange", "banana");
$replace = array("fruit1", "fruit2", "fruit3");

$newString = str_replace($search, $replace, $originalString);

// Display the new string
echo $newString;
?>

The output will be:

fruit1 fruit2 fruit3

Note that str_replace() is case-sensitive. If you need a case-insensitive replacement, you can use str_ireplace().
