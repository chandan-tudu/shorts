PHP Iterate Over An Array

In PHP, you can iterate over an array using various constructs. Here are some common ways to iterate over an array:

1. Using foreach Loop

The foreach loop is commonly used to iterate over each element of an array:

$fruits = array("apple", "banana", "orange", "grape");

foreach ($fruits as $fruit) {
    echo $fruit . "<br>";
}

This will output:

apple
banana
orange
grape

In the foreach loop, $fruits is the array, and $fruit is a variable that takes on the value of each element in the array in each iteration.

2. Using for Loop with count()

You can use a for loop along with the count() function to iterate over each element of the array by index:

$fruits = array("apple", "banana", "orange", "grape");
$arrayLength = count($fruits);

for ($i = 0; $i < $arrayLength; $i++) {
    echo $fruits[$i] . "<br>";
}

This will produce the same output as the foreach loop.

3. Using while Loop

You can also use a while loop to iterate over an array:

$fruits = array("apple", "banana", "orange", "grape");
$i = 0;

while ($i < count($fruits)) {
    echo $fruits[$i] . "<br>";
    $i++;
}

4. Using array_walk() Function

The array_walk() function can be used to apply a callback function to each element of an array:

$fruits = array("apple", "banana", "orange", "grape");

function printFruit($fruit) {
    echo $fruit . "<br>";
}

array_walk($fruits, 'printFruit');

These are just a few examples. The choice of which loop to use depends on the specific requirements of your code. The foreach loop is often the most convenient for iterating over the values of an array.
