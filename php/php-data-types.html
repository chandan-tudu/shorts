PHP Data Types

PHP supports several data types that are used to represent different kinds of values. Here is a list of common PHP data types:

  1. Integers:

    • Represent whole numbers without decimal points.
    • Example: $number = 42;

  2. Floating-Point Numbers (Floats):

    • Represent numbers with decimal points.
    • Example: $floatNumber = 3.14;

  3. Strings:

    • Represent sequences of characters.
    • Example: $string = "Hello, World!";

  4. Booleans:

    • Represent true or false values.
    • Example: $isTrue = true;

  5. Arrays:

    • Represent ordered maps that can contain other data types.
    • Example: $colors = array("red", "green", "blue");

  6. Objects:

    • Instances of user-defined classes, allowing the creation of custom data types.
    • Example:
    class Car {
    public $color = "red";
    public $brand = "Toyota";
}
$car = new Car();

  7. NULL:

    • Represents a variable with no value or a non-existent value.
    • Example: $variable = null;

  8. Resource:

    • Represents external resources (like database connections) that are not represented by a data type listed above.
    • Example: $file = fopen("example.txt", "r");

  9. Callable:

    • Represents a callable function or method.
    • Example:
    function myFunction() {
    echo "Hello, World!";
}
$functionName = 'myFunction';
$functionName();

  10. Iterable:

    • Represents an array or an object that can be iterated over using a foreach loop.
    • Introduced in PHP 7.1.
    • Example: $iterableArray = [1, 2, 3];

These examples illustrate the basic usage of each data type in PHP. Remember that PHP is a loosely typed language, so you don't need to explicitly declare the data type of a variable. The interpreter determines the data type based on the assigned value.
