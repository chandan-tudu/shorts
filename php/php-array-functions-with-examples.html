PHP Array Functions With Examples

PHP provides a variety of built-in functions to manipulate and work with arrays efficiently. Here are some commonly used array functions in PHP:

1. Array Creation:

  • array(): Create an array.

    $colors = array("Red", "Green", "Blue");

  • [] (Shorthand Syntax): Shorthand for creating arrays (available since PHP 5.4).

    $colors = ["Red", "Green", "Blue"];

2. Array Modification:

  • array_push(): Add one or more elements to the end of an array.

    $fruits = ["Apple", "Banana"];
array_push($fruits, "Orange", "Grapes");

  • array_pop(): Remove and return the last element of an array.

    $lastFruit = array_pop($fruits);

3. Array Manipulation:

  • array_merge(): Merge one or more arrays into a single array.

    $arr1 = [1, 2, 3];
$arr2 = [4, 5, 6];
$result = array_merge($arr1, $arr2);

  • array_slice(): Extract a slice of an array.

    $subset = array_slice($result, 2, 2);

4. Array Search:

  • in_array(): Checks if a value exists in an array.

    if (in_array("Green", $colors)) {
    // Found
}

  • array_search(): Searches an array for a given value and returns the corresponding key if successful.

    $key = array_search("Green", $colors);

5. Array Sorting:

  • sort(): Sort an array.

    sort($fruits);

  • rsort(): Sort an array in reverse order.

    rsort($fruits);

  • asort(): Sort an associative array by values.

    asort($person);

  • ksort(): Sort an associative array by keys.

    ksort($person);

6. Array Iteration:

  • foreach(): Loop through each key/value pair in an array.

    foreach ($fruits as $fruit) {
    echo $fruit . " ";
}

7. Array Functions for Statistics:

  • count(): Count the number of elements in an array.

    $count = count($fruits);

  • array_sum(): Calculate the sum of values in an array.

    $total = array_sum($numbers);

These are just a few examples of the many array functions available in PHP. You can explore the official PHP documentation for a comprehensive list and detailed explanations of each function.
