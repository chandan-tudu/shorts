PHP Array_map() Function

The array_map function in PHP is used to apply a given callback function to all elements of one or more arrays.

It returns an array containing the results of applying the callback to the corresponding elements of the input arrays. Here's the basic syntax:

array array_map ( callable $callback , array $array1 [, array $... ] )

  • $callback: The callback function to apply to each element of the input arrays. It can be any valid PHP callable, like a function or a method.

  • $array1: The array to which the callback will be applied.

  • $... (optional): Additional arrays. If provided, the callback function must accept as many arguments as there are arrays.

Here's a simple example:

<?php
// Example callback function
function square($n) {
    return $n * $n;
}

// Example array
$numbers = array(1, 2, 3, 4, 5);

// Apply the callback function to each element of the array
$squaredNumbers = array_map("square", $numbers);

// Display the result
print_r($squaredNumbers);
?>

Output:

Array
(
    [0] => 1
    [1] => 4
    [2] => 9
    [3] => 16
    [4] => 25
)

In this example, the square function is applied to each element of the $numbers array using array_map. The resulting array $squaredNumbers contains the squared values of each element.

You can also use anonymous functions (closures) with array_map:

<?php
// Example array
$numbers = array(1, 2, 3, 4, 5);

// Use an anonymous function to double each element
$doubledNumbers = array_map(function($n) {
    return $n * 2;
}, $numbers);

// Display the result
print_r($doubledNumbers);
?>

Output:

Array
(
    [0] => 2
    [1] => 4
    [2] => 6
    [3] => 8
    [4] => 10
)

In this case, the anonymous function is used to double each element of the $numbers array.
