PHP Comparison Operators With Examples

PHP comparison operators are used to compare two values and return a boolean result. Here are some common comparison operators along with examples and expected outputs:

1. Equal (==):

Checks if two values are equal.

$a = 5;
$b = "5";

$result = ($a == $b); // true

2. Not Equal (!=):

Checks if two values are not equal.

$a = 5;
$b = 10;

$result = ($a != $b); // true

3. Identical (===):

Checks if two values are equal and of the same type.

$a = 5;
$b = "5";

$result = ($a === $b); // false

4. Not Identical (!==):

Checks if two values are not equal or not of the same type.

$a = 5;
$b = "5";

$result = ($a !== $b); // true

5. Greater Than (>):

Checks if the left value is greater than the right value.

$a = 10;
$b = 5;

$result = ($a > $b); // true

6. Less Than (<):

Checks if the left value is less than the right value.

$a = 10;
$b = 15;

$result = ($a < $b); // true

7. Greater Than or Equal To (>=):

Checks if the left value is greater than or equal to the right value.

$a = 10;
$b = 10;

$result = ($a >= $b); // true

8. Less Than or Equal To (<=):

Checks if the left value is less than or equal to the right value.

$a = 5;
$b = 5;

$result = ($a <= $b); // true

9. Spaceship Operator (<=>):

Compares two values and returns 0 if they are equal, 1 if the left is greater, and -1 if the right is greater.

$a = 5;
$b = 10;

$result = ($a <=> $b); // Outputs: -1

These examples demonstrate how comparison operators work in PHP. The result of each operation is a boolean value (true or false). Keep in mind that the === and !== operators also consider the data type, not just the values.
