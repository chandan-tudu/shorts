PHP Magic Constants With Examples

PHP magic constants are special constants that change based on their usage. They provide dynamic information about the script at runtime.

These constants are prefixed with double underscores (__). Here are some of the commonly used PHP magic constants:

  1. __LINE__:

    • Returns the current line number of the file.
    echo __LINE__;  // Outputs the current line number

  2. __FILE__:

    • Returns the full path and filename of the file.
    echo __FILE__;  // Outputs the full path and filename of the script

  3. __DIR__:

    • Returns the directory of the file.
    echo __DIR__;  // Outputs the directory of the script

  4. __FUNCTION__:

    • Returns the name of the current function.
    function exampleFunction() {
    echo __FUNCTION__;  // Outputs the function name: exampleFunction
}

  5. __CLASS__:

    • Returns the name of the current class.
    class MyClass {
    public function showClassName() {
        echo __CLASS__;  // Outputs the class name: MyClass
    }
}

  6. __METHOD__:

    • Returns the name of the current method.
    class MyClass {
    public function exampleMethod() {
        echo __METHOD__;  // Outputs the method name: MyClass::exampleMethod
    }
}

  7. __NAMESPACE__:

    • Returns the name of the current namespace.
    namespace MyNamespace;

echo __NAMESPACE__;  // Outputs the namespace name: MyNamespace

These magic constants are helpful for debugging, logging, and obtaining information about the script's context at runtime. They are automatically populated by the PHP interpreter and do not require explicit definition. Keep in mind that the availability of some of these constants may vary depending on the PHP version and the context in which they are used.
