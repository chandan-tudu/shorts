How To Pass Function Arguments By Reference In PHP

In PHP, you can pass function arguments by reference by using the ampersand (&) before the parameter name in both the function definition and the function call. This means that any changes made to the parameter inside the function will affect the original variable outside the function.

Here's an example:

<?php
function increment(&$number) {
    $number++;
}

$counter = 5;

echo "Before: $counter<br>";

// Pass $counter by reference
increment($counter);

echo "After: $counter";
?>

In this example, the function increment takes a parameter $number by reference. When the function is called with $counter as an argument, any modifications made to $number inside the function will directly affect the original variable $counter outside the function.

Output:

Before: 5
After: 6

Key points:

  1. The & symbol before the parameter in the function definition indicates that it is passed by reference.

  2. When calling the function, you don't need to use the & symbol again; it's only used in the function definition.

It's important to use this feature judiciously, as passing variables by reference can lead to unexpected behavior if not handled carefully. Generally, it's recommended to pass arguments by value whenever possible, and only use passing by reference when necessary.
