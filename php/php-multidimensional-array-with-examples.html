PHP Multidimensional Array With Examples

In PHP, a multidimensional array is an array that contains one or more arrays as its elements.

Each inner array can have its own set of keys and values, effectively creating a two-dimensional or multidimensional structure. Here's how you can create, access, and manipulate multidimensional arrays in PHP:

1. Creating a Multidimensional Array:

// Creating a simple two-dimensional array
$matrix = array(
    array(1, 2, 3),
    array(4, 5, 6),
    array(7, 8, 9)
);

You can also use the shorthand syntax introduced in PHP 5.4:

// Shorthand syntax
$matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

2. Accessing Elements:

You access elements in a multidimensional array using indices for both the outer and inner arrays:

echo $matrix[0][0]; // Outputs: 1
echo $matrix[1][2]; // Outputs: 6

3. Modifying Elements:

You can modify the value of an existing element:

$matrix[1][1] = 10;
echo $matrix[1][1]; // Outputs: 10

4. Adding Elements:

You can add new elements to a multidimensional array:

$matrix[2][] = 11;
$matrix[2][] = 12;
$matrix[2][] = 13;

5. Looping Through Multidimensional Arrays:

You can use nested foreach loops to iterate through all elements of a multidimensional array:

foreach ($matrix as $row) {
    foreach ($row as $value) {
        echo $value . " ";
    }
    echo "<br>";
}
// Outputs:
// 1 2 3
// 4 10 6
// 7 8 9 11 12 13

Multidimensional arrays are useful when you need to represent tabular or matrix-like data. They are commonly used in scenarios such as representing tables, matrices, and data structures with multiple levels of hierarchy.
