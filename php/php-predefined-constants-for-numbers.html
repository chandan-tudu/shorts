PHP Predefined Constants For Numbers

PHP has several predefined constants related to numbers, mathematics, and general numeric values. Here are some commonly used ones:

  1. Mathematical Constants:

    • M_PI: Represents the mathematical constant Ï€ (pi).
    • M_E: Represents the mathematical constant e.
    echo M_PI;  // Outputs: 3.1415926535898
echo M_E;   // Outputs: 2.718281828459

  2. Integer Constants:

    • PHP_INT_MAX: Represents the maximum value for integers on your system.
    • PHP_INT_MIN: Represents the minimum value for integers on your system.
    echo PHP_INT_MAX;  // Outputs the maximum integer value on your system
echo PHP_INT_MIN;  // Outputs the minimum integer value on your system

  3. Float Constants:

    • PHP_FLOAT_MAX: Represents the maximum representable finite float.
    • PHP_FLOAT_MIN: Represents the minimum representable positive float.
    echo PHP_FLOAT_MAX;  // Outputs the maximum representable finite float
echo PHP_FLOAT_MIN;  // Outputs the minimum representable positive float

  4. Infinity Constants:

    • INF: Represents positive infinity.
    • -INF: Represents negative infinity.
    echo INF;   // Outputs: INF
echo -INF;  // Outputs: -INF

  5. NaN (Not a Number) Constant:

    • NAN: Represents "Not a Number," typically the result of an undefined or unrepresentable mathematical operation.
    echo NAN;  // Outputs: NAN

These constants are part of the PHP language and are often used in mathematical calculations, comparisons, or for representing special numeric values. Keep in mind that some of these constants may vary depending on your PHP version and the platform on which PHP is running.
