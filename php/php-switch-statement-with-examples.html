PHP Switch Statement With Examples

The switch statement in PHP is used for making decisions based on the value of an expression. It provides an alternative way to handle multiple conditions compared to the if-elseif-else statements. Here's the basic syntax of the switch statement:

switch (expression) {
    case value1:
        // Code to be executed if expression matches value1
        break;
    case value2:
        // Code to be executed if expression matches value2
        break;
    // Additional cases as needed
    default:
        // Code to be executed if none of the cases match
}

Each case represents a possible value of the expression, and the code within the matching case is executed. If none of the cases match, the code within the default block is executed (if provided).

Example:

$day = "Monday";

switch ($day) {
    case "Monday":
        echo "It's the start of the week.";
        break;
    case "Friday":
        echo "It's the end of the week.";
        break;
    default:
        echo "It's a regular day.";
}

In this example:

  • If the value of $day is "Monday", it prints "It's the start of the week."
  • If the value of $day is "Friday", it prints "It's the end of the week."
  • If the value of $day doesn't match any of the cases, it prints "It's a regular day."

Note that each case block ends with a break statement. This is important because it prevents the execution of subsequent case blocks. Without the break statement, code execution would "fall through" to the next case.

Also, the default case is optional. If it's not provided and none of the cases match, no code will be executed.

The switch statement is particularly useful when you have a single expression to compare against multiple possible values.
