PHP Break And Continue Statements With Examples

In PHP, the break and continue statements are used within loops to control the flow of execution.

1. "break" Statement:

The break statement is used to exit a loop prematurely. It is often used to terminate a loop based on a certain condition.

Example:

for ($i = 1; $i <= 10; $i++) {
    if ($i == 5) {
        break; // Exit the loop when $i is 5
    }
    echo $i . " ";
}
// Outputs: 1 2 3 4

In this example, the for loop iterates from 1 to 10, but when $i becomes equal to 5, the break statement is executed, and the loop is terminated.

2. "continue" Statement:

The continue statement is used to skip the rest of the code in the current iteration of a loop and move on to the next iteration.

Example:

for ($i = 1; $i <= 10; $i++) {
    if ($i % 2 == 0) {
        continue; // Skip even numbers
    }
    echo $i . " ";
}
// Outputs: 1 3 5 7 9

In this example, the for loop iterates from 1 to 10, but when $i is an even number, the continue statement is executed, skipping the echo statement and moving on to the next iteration.

These statements provide control over the loop's behavior, allowing you to customize the execution flow based on specific conditions. The break statement is used to exit the entire loop, while the continue statement is used to skip the rest of the code in the current iteration and move to the next one.
