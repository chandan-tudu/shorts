PHP Ternary Operator With Example

The ternary operator is a shorthand way of writing an if-else statement in PHP. It provides a concise syntax for evaluating a condition and returning different values based on whether the condition is true or false.

Syntax:

$variable = (condition) ? value_if_true : value_if_false;

The expression (condition) ? value_if_true : value_if_false evaluates the condition. If the condition is true, it assigns value_if_true to the variable; otherwise, it assigns value_if_false.

Example:

$age = 25;

// Ternary operator to determine if the person is an adult
$isAdult = ($age >= 18) ? "Yes" : "No";
echo $isAdult;  // Outputs: Yes

In this example, if the condition $age >= 18 is true, the variable $isAdult will be assigned the value "Yes"; otherwise, it will be assigned the value "No".

Nested Ternary Operators:

You can also nest ternary operators, although it's important to maintain readability to avoid confusion.

$marks = 75;

// Nested ternary operators to determine the grade
$grade = ($marks >= 90) ? "A" : (($marks >= 80) ? "B" : (($marks >= 70) ? "C" : "D"));
echo $grade;  // Outputs: C

Here, the nested ternary operators are used to determine the grade based on the marks.

While ternary operators can make code more concise, it's essential to use them judiciously to maintain readability. In some cases, using a traditional if-else statement might be more appropriate, especially for complex conditions or multiple statements.
