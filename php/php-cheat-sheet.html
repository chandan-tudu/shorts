PHP Cheat Sheet

Here's a concise PHP cheat sheet covering some of the commonly used syntax and functions:

Basic Syntax:

<?php
// Comments
// Single-line comment
/*
   Multi-line comment
*/

// Variables
$variable = "value";

// Data Types
$string = "Hello";
$integer = 123;
$float = 3.14;
$boolean = true;
$array = array(1, 2, 3);
$associativeArray = array("key" => "value");

// Constants
define("PI", 3.14);

// Operators
$result = $a + $b;
$result = $a - $b;
$result = $a * $b;
$result = $a / $b;
$result = $a % $b;
$result = $a == $b;
$result = $a === $b;
$result = $a != $b;
$result = $a !== $b;

// Control Structures
if (condition) {
    // code to execute if the condition is true
} elseif (anotherCondition) {
    // code to execute if anotherCondition is true
} else {
    // code to execute if none of the conditions are true
}

switch ($variable) {
    case 'value1':
        // code for value1
        break;
    case 'value2':
        // code for value2
        break;
    default:
        // code for default case
        break;
}

// Loops
for ($i = 0; $i < 5; $i++) {
    // code to repeat
}

while (condition) {
    // code to repeat while the condition is true
}

do {
    // code to repeat at least once, then check the condition
} while (condition);

foreach ($array as $value) {
    // code to execute for each element in the array
}

// Functions
function functionName($parameter1, $parameter2) {
    // code to execute
    return $result;
}

// Include and Require
include "filename.php";
require "filename.php";
?>

Strings:

// Concatenation
$fullName = $firstName . " " . $lastName;

// String Length
$length = strlen($str);

// Substring
$substring = substr($str, $start, $length);

// String Replace
$newStr = str_replace("old", "new", $str);

// String to Lower/Upper Case
$lowercase = strtolower($str);
$uppercase = strtoupper($str);

// Trim Whitespace
$trimmedStr = trim($str);

Arrays:

// Indexed Array
$colors = array("red", "green", "blue");

// Associative Array
$person = array("name" => "John", "age" => 30);

// Multidimensional Array
$matrix = array(
    array(1, 2, 3),
    array(4, 5, 6)
);

// Accessing Array Elements
$firstColor = $colors[0];
$personName = $person["name"];
$value = $matrix[1][2];

// Array Functions
$length = count($array);
$sortedArray = sort($array);
$reversedArray = array_reverse($array);

Dates and Times:

// Current Date and Time
$currentDateTime = date("Y-m-d H:i:s");

// Custom Date and Time Formatting
$formattedDate = date("F j, Y", strtotime("2023-12-25"));

// Timezone Setting
date_default_timezone_set("UTC");

This cheat sheet covers a range of basic PHP concepts. Remember that PHP is a powerful and versatile language with many features, and this cheat sheet is just a starting point. For more in-depth information, refer to the official PHP documentation at php.net.
