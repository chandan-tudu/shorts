PHP Operators With Examples

PHP supports a variety of operators that are used to perform operations on variables and values. Here are some of the common types of operators in PHP along with examples:

1. Arithmetic Operators:

Perform basic arithmetic operations.

$a = 10;
$b = 5;

$sum = $a + $b;      // Addition
$diff = $a - $b;     // Subtraction
$prod = $a * $b;     // Multiplication
$quotient = $a / $b;  // Division
$remainder = $a % $b; // Modulus (remainder after division)

2. Assignment Operators:

Assign values to variables.

$x = 5;
$y = 10;

$x += $y;  // Equivalent to $x = $x + $y;
$x -= $y;  // Equivalent to $x = $x - $y;
$x *= $y;  // Equivalent to $x = $x * $y;
$x /= $y;  // Equivalent to $x = $x / $y;
$x %= $y;  // Equivalent to $x = $x % $y;

3. Comparison Operators:

Compare values and return a boolean result.

$p = 10;
$q = 5;

$isEqual = ($p == $q);     // Equal
$isNotEqual = ($p != $q);  // Not Equal
$isIdentical = ($p === $q); // Identical (value and type)
$isNotIdentical = ($p !== $q); // Not Identical
$isGreaterThan = ($p > $q); // Greater Than
$isLessThan = ($p < $q);    // Less Than

4. Logical Operators:

Perform logical operations and return a boolean result.

$condition1 = true;
$condition2 = false;

$andResult = $condition1 && $condition2; // AND
$orResult = $condition1 || $condition2;  // OR
$notResult = !$condition1;                // NOT

5. Increment and Decrement Operators:

Increment or decrement a variable's value.

$counter = 5;

$counter++; // Increment by 1
$counter--; // Decrement by 1

6. String Operators:

Concatenate strings.

$firstName = "John";
$lastName = "Doe";

$fullName = $firstName . " " . $lastName; // Concatenation

7. Ternary Operator:

Shortens the if-else statement.

$age = 18;

$isAdult = ($age >= 18) ? "Yes" : "No";

8. Null Coalescing Operator:

Returns the first defined and non-null value.

$username = $_GET['user'] ?? 'Guest';

These are just a few examples of the many operators available in PHP. Operators play a crucial role in performing various operations and comparisons within your PHP code.
