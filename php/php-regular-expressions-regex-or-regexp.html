PHP Regular Expressions (Regex Or Regexp)

Regular expressions (regex or regexp) in PHP are powerful tools for pattern matching and manipulation of strings. PHP provides a set of functions for working with regular expressions, primarily using the PCRE (Perl-Compatible Regular Expressions) library.

Here are some common PHP functions related to regular expressions:

  1. preg_match(): Checks if a pattern exists in a string.

    $pattern = '/hello/';
$string = 'Hello, world!';
if (preg_match($pattern, $string)) {
    echo 'Pattern found!';
} else {
    echo 'Pattern not found.';
}
// Output: Pattern found!

  2. preg_match_all(): Similar to preg_match() but finds all matches in the string.

    $pattern = '/\d+/';
$string = 'There are 123 cats and 456 dogs.';
preg_match_all($pattern, $string, $matches);
print_r($matches);
// Output: Array ( [0] => Array ( [0] => 123 [1] => 456 ) )

  3. preg_replace(): Searches a string for a pattern and replaces it with another string.

    $pattern = '/\bapples\b/';
$string = 'I like apples and oranges.';
$replacement = 'bananas';
$result = preg_replace($pattern, $replacement, $string);
echo $result;
// Output: I like bananas and oranges.

  4. preg_split(): Splits a string into an array using a regular expression.

    $pattern = '/\s+/';
$string = 'Apple Banana Cherry';
$result = preg_split($pattern, $string);
print_r($result);
// Output: Array ( [0] => Apple [1] => Banana [2] => Cherry )

  5. Modifiers: Regular expressions often use modifiers to change the behavior. Common ones include:

    • i: Case-insensitive matching.
    • m: Treats the string as multiple lines.
    • s: Treats the string as a single line.
    • u: Enables UTF-8 matching.
    $pattern = '/apple/i';
$string = 'I like Apple.';
if (preg_match($pattern, $string)) {
    echo 'Pattern found!';
} else {
    echo 'Pattern not found.';
}
// Output: Pattern found!

  6. Character Classes: Character classes allow you to match one of a set of characters.

    • [0-9]: Matches any digit.
    • [a-zA-Z]: Matches any uppercase or lowercase letter.
    $pattern = '/[0-9]/';
$string = 'There are 123 cats.';
if (preg_match($pattern, $string)) {
    echo 'Digit found!';
} else {
    echo 'No digit found.';
}
// Output: Digit found!

These are just basic examples, and regular expressions can become quite complex depending on your needs. Regular expressions are a powerful tool, but they can also be challenging to master. It's often helpful to use online regex testers and resources to build and test your regular expressions.
