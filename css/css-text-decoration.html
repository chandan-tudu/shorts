CSS Text Decoration

In CSS (Cascading Style Sheets), the text-decoration property is used to control the decoration of text, such as underlining, overlining, or striking through. Here's how you can use it:

/* Example 1: Underline */
.text-underlined {
    text-decoration: underline;
}

/* Example 2: Overline */
.text-overlined {
    text-decoration: overline;
}

/* Example 3: Line-through (strikethrough) */
.text-strikethrough {
    text-decoration: line-through;
}

/* Example 4: Blink (not widely supported) */
.text-blink {
    text-decoration: blink;
}

/* Example 5: Multiple decorations */
.text-multiple {
    text-decoration: underline overline line-through;
}

/* Example 6: Removing decoration */
.text-no-decoration {
    text-decoration: none;
}

In the above examples:

  • text-decoration: underline; underlines the text.
  • text-decoration: overline; puts a line above the text.
  • text-decoration: line-through; creates a strikethrough effect.
  • text-decoration: blink; makes the text blink, but note that the blink value is not widely supported and may not work in all browsers.
  • text-decoration: none; removes any text decoration.

You can apply these styles to HTML elements like this:

<p class="text-underlined">This text is underlined.</p>
<p class="text-overlined">This text has an overline.</p>
<p class="text-strikethrough">This text has a strikethrough.</p>
<p class="text-blink">This text blinks (if supported).</p>
<p class="text-multiple">This text has multiple decorations.</p>
<p class="text-no-decoration">This text has no decoration.</p>
