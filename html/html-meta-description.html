HTML Meta Description

In HTML, the meta description is a meta tag that provides a concise and brief summary of the content of a web page.

It is not displayed on the page itself but is often used by search engines to provide a preview snippet in search results.

The meta description is placed within the <head> section of an HTML document. Here's an example of how to add a meta description to an HTML document:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8" />
        <meta
            name="description"
            content="This is a brief description of your web page content. It should be concise and informative."
        />
        <meta name="keywords" content="keywords, related, to, your, content" />
        <meta name="author" content="Your Name" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Your Page Title</title>
    </head>
    <body>
        <!-- Your page content goes here -->
    </body>
</html>

In the example above:

  • <meta name="description" content="...">: This is the meta tag for the description. Replace the content attribute with a concise and relevant description of your web page.

  • <meta name="keywords" content="...">: This meta tag used to be more important for search engines, but it is less influential today. However, you can still include relevant keywords separated by commas.

  • <meta name="author" content="...">: This meta tag allows you to specify the author of the web page.

  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: This meta tag is often included for responsive web design, ensuring that the page scales properly on different devices.

Remember that search engines may use or ignore the meta description, and the length that is displayed in search results may be limited.

Aim for a clear and informative description within the specified length guidelines of the search engines you are targeting.
