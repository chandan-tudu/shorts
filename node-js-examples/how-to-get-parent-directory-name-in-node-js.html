How To Get Parent Directory Name In Node.js

In Node.js, you can use the path module to get the parent directory name by combining the path.dirname() method with the path.basename() method. Here's an example:

const path = require('path');

// Example file path
const filePath = '/path/to/your/file.txt';

// Use the path module to get the parent directory name
const parentDirectoryName = path.basename(path.dirname(filePath));

// Print the parent directory name
console.log('Parent Directory Name:', parentDirectoryName); // Output: your

In this example, path.dirname(filePath) extracts the directory name from the file path, and then path.basename() is used to get the base name (last portion) of that directory path, which effectively gives you the parent directory name.

If you have a directory path and want to get its parent directory name, you can do the following:

const path = require('path');

// Example directory path
const directoryPath = '/path/to/your/directory/';

// Use the path module to get the parent directory name
const parentDirectoryName = path.basename(path.dirname(directoryPath));

// Print the parent directory name
console.log('Parent Directory Name:', parentDirectoryName);

This approach works for both file paths and directory paths. It may be useful to note that the result of path.basename() will not include the trailing slash if the path is a directory. If you want to include the trailing slash, you can append it manually.
