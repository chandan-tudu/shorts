How To Get The Last Portion Of A Path In Node.js

To get the last portion of a path in Node.js, you can use the path.basename() method from the path module. Here's an example:

const path = require('path');

// Example file path
const filePath = '/path/to/your/file.txt';

// Use the path module to get the last portion of the path
const lastPortion = path.basename(filePath);

// Print the last portion
console.log('Last Portion:', lastPortion); // Output: file.txt

In this example, path.basename(filePath) extracts the last portion of the file path, which is the file or directory name. This method is useful for getting the base name of a file or directory path.

If you have a directory path with a trailing slash and you want to include that slash in the last portion, you can set the second parameter of path.basename() to the character you want to treat as the path separator:

const path = require('path');

// Example directory path with a trailing slash
const directoryPath = '/path/to/your/directory/';

// Use the path module to get the last portion of the path with the trailing slash
const lastPortionWithSlash = path.basename(directoryPath, '/');

// Print the last portion with the trailing slash
console.log('Last Portion with Slash:', lastPortionWithSlash); // Output: directory

In this case, path.basename(directoryPath, '/') ensures that the trailing slash is included in the last portion of the path.
