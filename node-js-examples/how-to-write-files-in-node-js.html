How To Write Files In Node.js

In Node.js, you can write files using the fs (file system) module. Below are examples of how to write files both synchronously and asynchronously.

Synchronous File Writing:

const fs = require('fs');

const dataToWrite = 'Hello, this is some content to write to the file.';

try {
    // Write the file synchronously
    fs.writeFileSync('example.txt', dataToWrite, 'utf8');
    console.log('File written successfully!');
} catch (error) {
    console.error('Error writing file:', error);
}

In this example, writeFileSync is a synchronous method that writes data to a file. If the file already exists, it will be overwritten. If an error occurs during the write operation, it is caught and logged.

Asynchronous File Writing:

const fs = require('fs');

const dataToWrite = 'Hello, this is some content to write to the file.';

// Write the file asynchronously
fs.writeFile('example.txt', dataToWrite, 'utf8', (err) => {
    if (err) {
        console.error('Error writing file:', err);
        return;
    }
    console.log('File written successfully!');
});

In the asynchronous example, writeFile is used, which takes a callback function as its last argument. The callback is called once the file is written, and it receives an error (if any).

Remember to replace 'example.txt' with the actual path and filename you want to write to, and adjust the dataToWrite variable to contain the content you want to write to the file.

Asynchronous file operations are generally recommended to avoid blocking the event loop, especially in applications with many concurrent operations.
