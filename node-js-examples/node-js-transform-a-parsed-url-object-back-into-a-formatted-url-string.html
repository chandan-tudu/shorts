Node.js - Transform A Parsed URL Object Back Into A Formatted URL String

In Node.js, you can use the url.format() method to transform a parsed URL object back into a formatted URL string. Here's an example:

Using url.parse (deprecated in recent Node.js versions):

const url = require('url');

// Example URL
const urlString = 'https://www.example.com/path?query=123#fragment';

// Parse the URL
const parsedUrl = url.parse(urlString, true);

// Modify the parsed URL if needed
parsedUrl.query.newParam = '456';

// Transform the parsed URL back into a formatted URL string
const formattedUrl = url.format(parsedUrl);

console.log('Original URL:', urlString);
console.log('Formatted URL:', formattedUrl);

Output:

Original URL: https://www.example.com/path?query=123#fragment
Formatted URL: https://www.example.com/path?query=123&newParam=456#fragment

Using new URL constructor:

const { URL } = require('url');

// Example URL
const urlString = 'https://www.example.com/path?query=123#fragment';

// Parse the URL using the URL constructor
const parsedUrl = new URL(urlString);

// Modify the parsed URL if needed
parsedUrl.searchParams.set('newParam', '456');

// Transform the parsed URL back into a formatted URL string
const formattedUrl = parsedUrl.toString();

console.log('Original URL:', urlString);
console.log('Formatted URL:', formattedUrl);

Output:

Original URL: https://www.example.com/path?query=123#fragment
Formatted URL: https://www.example.com/path?query=123&newParam=456#fragment

In both cases, the url.format() method or URL.toString() method is used to convert the parsed URL object back into a formatted URL string. The resulting formattedUrl will contain the modifications made to the parsed URL object, if any.
