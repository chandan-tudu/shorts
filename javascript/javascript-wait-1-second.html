JavaScript Wait 1 Second

In JavaScript, you can use the setTimeout function to introduce a delay or wait for a specific amount of time.

If you want to wait for 1 second (1000 milliseconds), you can use the following example:

console.log('Start');

// Use setTimeout to wait for 1 second (1000 milliseconds)
setTimeout(function () {
    console.log('Waited for 1 second');
}, 1000);

console.log('End');

In this example:

  • The console.log("Start") statement is executed first.
  • The setTimeout function is used to delay the execution of the provided function by 1000 milliseconds (1 second).
  • Inside the setTimeout, console.log("Waited for 1 second") will be executed after the 1-second delay.
  • Finally, console.log("End") is executed.

Keep in mind that JavaScript is single-threaded, and setTimeout doesn't pause the entire script. Instead, it schedules the provided function to be executed after the specified delay. Other code can continue running in the meantime.

If you need to perform an action after a delay without blocking the main thread, using setTimeout or setInterval is a common approach.
