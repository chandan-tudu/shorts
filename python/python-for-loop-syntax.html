Python "for" Loop Syntax

In Python, the for loop is used to iterate over a sequence (such as a list, tuple, string, or range) or any other iterable object. The basic syntax of a for loop in Python is as follows:

for variable in iterable:
    # Code to be executed in each iteration
    # ...

# Code outside the loop (not indented) runs after the loop completes
# ...

Here's a simple example using a list:

# Iterate over a list
my_list = [1, 2, 3, 4, 5]

for number in my_list:
    print(number)

This loop will iterate over each element in my_list, and in each iteration, the value of the current element will be assigned to the variable number, and the print(number) statement will be executed.

You can also use the range() function to generate a sequence of numbers for the loop:

# Iterate over a range of numbers
for i in range(1, 6):
    print(i)

In this example, the range(1, 6) generates numbers from 1 to 5 (inclusive), and the loop prints each number in the sequence.

Remember that the indentation is crucial in Python. The code within the loop must be indented to indicate the block of code that belongs to the loop. The loop continues until it has iterated over all items in the iterable.

It's important to note that Python does not use a counter variable to control the loop, as you might see in some other programming languages. Instead, Python directly iterates over the elements of the iterable. If you need both the index and the value, you can use the enumerate() function:

# Iterate over a list with both index and value
for index, value in enumerate(my_list):
    print(f"Index: {index}, Value: {value}")
