Python Strings Methods

In Python, strings are objects, and there are numerous built-in methods that you can use to perform various operations on strings. Here are some common string methods:

  1. len() method:

    • Returns the length of the string (number of characters).
    my_string = "Hello, World!"
length = len(my_string)
print(length)  # Output: 13

  2. lower() and upper() methods:

    • Converts all characters in a string to lowercase or uppercase, respectively.
    my_string = "Python Programming"
lower_case = my_string.lower()
upper_case = my_string.upper()
print(lower_case)  # Output: "python programming"
print(upper_case)  # Output: "PYTHON PROGRAMMING"

  3. strip(), lstrip(), and rstrip() methods:

    • strip() Removes leading and trailing whitespaces from a string. lstrip() removes leading (left) whitespaces, and rstrip() removes trailing (right) whitespaces.
    my_string = "   Python Programming   "
stripped_string = my_string.strip()
print(stripped_string)  # Output: "Python Programming"

  4. replace(old, new) method:

    • Replaces occurrences of the specified old substring with the new substring.
    my_string = "Hello, World!"
new_string = my_string.replace("Hello", "Hi")
print(new_string)  # Output: "Hi, World!"

  5. find(substring) and index(substring) methods:

    • Both methods return the index of the first occurrence of the specified substring. The main difference is that find() returns -1 if the substring is not found, while index() raises a ValueError.
    my_string = "Python Programming"
index1 = my_string.find("Pro")
index2 = my_string.index("Pro")
print(index1)  # Output: 7
print(index2)  # Output: 7

  6. count(substring) method:

    • Returns the number of occurrences of the specified substring in the string.
    my_string = "programming is fun, and programming is creative."
count = my_string.count("programming")
print(count)  # Output: 2

  7. startswith(prefix) and endswith(suffix) methods:

    • Checks if the string starts or ends with the specified prefix or suffix, respectively.
    my_string = "Python Programming"
starts_with_python = my_string.startswith("Python")
ends_with_ing = my_string.endswith("ing")
print(starts_with_python)  # Output: True
print(ends_with_ing)        # Output: True

These are just a few examples of the many string methods available in Python. Understanding and utilizing these methods will make it easier to manipulate and process strings in your Python programs.
