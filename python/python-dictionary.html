Python Dictionary

In Python, a dictionary is a mutable, unordered collection of key-value pairs, where each key must be unique.

Dictionaries are defined using curly braces {} and consist of key-value pairs separated by colons. Here's a basic example:

# Creating a dictionary
my_dict = {'name': 'John', 'age': 30, 'city': 'New York'}

# Accessing values using keys
print("Name:", my_dict['name'])
print("Age:", my_dict['age'])
print("City:", my_dict['city'])

In this example, my_dict is a dictionary with keys 'name', 'age', and 'city', each associated with a corresponding value.

You can add, modify, or remove key-value pairs in a dictionary:

# Adding a new key-value pair
my_dict['occupation'] = 'Engineer'

# Modifying the value of an existing key
my_dict['age'] = 31

# Removing a key-value pair
del my_dict['city']

# Displaying the updated dictionary
print("Updated Dictionary:", my_dict)

Dictionaries support various operations and methods. Some common operations include checking if a key exists, getting a list of keys or values, and checking the length of the dictionary:

# Checking if a key exists
if 'age' in my_dict:
    print("Age is present in the dictionary")

# Getting a list of keys and values
keys = list(my_dict.keys())
values = list(my_dict.values())

print("Keys:", keys)
print("Values:", values)

# Checking the length of the dictionary
length = len(my_dict)
print("Length of the dictionary:", length)

Dictionaries are versatile and widely used in Python for tasks such as storing and retrieving configuration settings, representing JSON-like data structures, and more. They provide a convenient way to associate data with descriptive labels (keys).
