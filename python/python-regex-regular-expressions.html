Python RegEx (Regular Expressions)

Regular expressions, commonly referred to as regex or regexp, are sequences of characters that define a search pattern. They are used for string manipulation and searching. In Python, the re module provides support for regular expressions.

Here's a brief overview of some common regular expression operations in Python:

1. Matching:

  • re.match(pattern, string): Checks if the pattern matches at the beginning of the string.
import re

pattern = r"Hello"
string = "Hello, World!"

match_result = re.match(pattern, string)

if match_result:
    print("Match found!")
else:
    print("No match.")

2. Searching:

  • re.search(pattern, string): Searches for the pattern anywhere in the string.
import re

pattern = r"World"
string = "Hello, World!"

search_result = re.search(pattern, string)

if search_result:
    print("Pattern found!")
else:
    print("Pattern not found.")

3. Substitution:

  • re.sub(pattern, replacement, string): Replaces occurrences of the pattern with the specified replacement in the string.
import re

pattern = r"\d+"  # Matches one or more digits
string = "The price is $20 and $30."

replacement = "XX"
result = re.sub(pattern, replacement, string)

print(result)  # Output: "The price is $XX and $XX."

4. Groups:

You can use groups to capture specific parts of the matched pattern.

import re

pattern = r"(\d+)\s+(\w+)"
string = "Price: 25 USD"

match_result = re.match(pattern, string)

if match_result:
    number = match_result.group(1)
    currency = match_result.group(2)
    print(f"Number: {number}, Currency: {currency}")
else:
    print("No match.")

5. Flags:

Flags can be used to modify the behavior of regular expression patterns. For example, the re.IGNORECASE flag makes the pattern case-insensitive.

import re

pattern = r"hello"
string = "Hello, World!"

match_result = re.search(pattern, string, re.IGNORECASE)

if match_result:
    print("Pattern found!")
else:
    print("Pattern not found.")

These are just basic examples, and regular expressions can become quite complex depending on the requirements. The official Python documentation for the re module provides detailed information and examples: Python re module documentation.
