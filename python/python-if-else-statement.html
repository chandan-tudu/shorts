Python If...else Statement

In Python, the if-else statement is used for conditional execution. It allows you to execute a block of code if a certain condition is true and another block of code if the condition is false. The basic syntax is as follows:

if condition:
    # Code to execute if the condition is true
else:
    # Code to execute if the condition is false

Here's an example:

x = 10

if x > 5:
    print("x is greater than 5")
else:
    print("x is not greater than 5")

Output:

x is greater than 5

In this example, if the condition x > 5 is true, the first block of code (indented under if) will be executed, printing "x is greater than 5." Otherwise, the block of code under else will be executed, printing "x is not greater than 5."

You can also have multiple conditions using elif (short for "else if"):

y = 20

if y > 50:
    print("y is greater than 50")
elif y > 10:
    print("y is greater than 10 but not greater than 50")
else:
    print("y is 10 or less")

Output:

y is greater than 10 but not greater than 50

In this example, the first condition checks if y > 50, and if true, the corresponding block of code is executed. If that condition is false, it checks the next condition (elif y > 10) and executes the corresponding block of code if true. If none of the conditions is true, the block of code under else is executed.

✍️ Note: Indentation is crucial in Python. The blocks of code under if, elif, and else must be indented consistently to indicate their scope.
