Python "random" Module

In Python, the random module provides functions for generating random numbers. To use the random module, you need to import it first. Here are some common functions provided by the random module:

  1. Importing the random module:

    import random

  2. Generating Random Numbers:

    • random.random(): Returns a random floating-point number in the range (0.0, 1.0).

      result = random.random()
print(result)  # Output: a random float between 0.0 and 1.0 (exclusive)

    • random.uniform(a, b): Returns a random floating-point number in the range (a, b).

      result = random.uniform(1, 10)
print(result)  # Output: a random float between 1 and 10 (exclusive)

    • random.randint(a, b): Returns a random integer in the range [a, b] (inclusive).

      result = random.randint(1, 6)
print(result)  # Output: a random integer between 1 and 6 (inclusive)

    • random.choice(sequence): Returns a random element from the given sequence (list, tuple, or string).

      my_list = [1, 2, 3, 4, 5]
result = random.choice(my_list)
print(result)  # Output: a random element from the list

  3. Shuffling Sequences:

    • random.shuffle(sequence): Shuffles the elements of the sequence in place.

      my_list = [1, 2, 3, 4, 5]
random.shuffle(my_list)
print(my_list)  # Output: the elements of the list in a random order

  4. Random Sampling:

    • random.sample(sequence, k): Returns a new list containing k unique elements randomly chosen from the given sequence.

      my_list = [1, 2, 3, 4, 5]
result = random.sample(my_list, 3)
print(result)  # Output: a list with 3 unique elements randomly chosen from my_list

These are just a few examples of the functions provided by the random module. The module also includes functions for seeding the random number generator (random.seed()), generating random integers with a specific step (random.randrange()), and more. For more details, you can refer to the official documentation.
