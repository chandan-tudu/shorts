Python Datetime Module

The datetime module in Python provides classes for working with dates and times. It includes various classes, such as datetime, date, time, timedelta, and timezone, to represent and manipulate date and time information.

Here's an overview of some key classes and functions in the datetime module:

1. datetime Class:

The datetime class is used to represent date and time information.

from datetime import datetime

# Current date and time
now = datetime.now()
print("Current date and time:", now)

# Creating a specific datetime object
specific_date = datetime(2022, 12, 31, 18, 30, 0)
print("Specific date and time:", specific_date)

2. date and time Classes:

The date class represents a date (year, month, day), and the time class represents a time (hour, minute, second, microsecond).

from datetime import date, time

# Creating date and time objects
my_date = date(2022, 12, 31)
my_time = time(18, 30, 0)

print("Date:", my_date)
print("Time:", my_time)

3. timedelta Class:

The timedelta class represents the duration between two dates or times.

from datetime import timedelta

# Creating timedelta objects
delta = timedelta(days=5, hours=3, minutes=30)
print("Timedelta:", delta)

# Performing operations with timedelta
new_date = datetime.now() + delta
print("New date:", new_date)

4. Formatting and Parsing:

The strftime method allows you to format a datetime object as a string, and the strptime function allows you to parse a string into a datetime object.

from datetime import datetime

# Formatting a datetime object
now = datetime.now()
formatted_date = now.strftime("%Y-%m-%d %H:%M:%S")
print("Formatted date:", formatted_date)

# Parsing a string into a datetime object
date_string = "2022-12-31 18:30:00"
parsed_date = datetime.strptime(date_string, "%Y-%m-%d %H:%M:%S")
print("Parsed date:", parsed_date)

5. Timezone Support:

The timezone class allows you to represent time zones, and you can use the astimezone method to convert a datetime object from one time zone to another.

from datetime import datetime, timezone, timedelta

# Creating timezone objects
utc = timezone.utc
eastern = timezone(timedelta(hours=-5))

# Getting current time in UTC
now_utc = datetime.now(utc)
print("Current time in UTC:", now_utc)

# Converting to Eastern time
now_eastern = now_utc.astimezone(eastern)
print("Current time in Eastern time:", now_eastern)

These are just some basic examples, and the datetime module provides many more functionalities for working with dates and times. Refer to the official Python documentation for detailed information: Python datetime module documentation.
