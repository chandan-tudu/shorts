Python Class Inheritance (OOP)

Inheritance is a fundamental concept in object-oriented programming (OOP) that allows a new class to inherit attributes and methods from an existing class. The existing class is called the base class or parent class, and the new class is called the derived class or child class. Inheritance promotes code reuse and the creation of a hierarchy of classes. Here's an overview of how inheritance works in Python:

Syntax of Inheritance:

class BaseClass:
    # Attributes and methods of the base class

class DerivedClass(BaseClass):
    # Additional attributes and methods specific to the derived class

Example of Inheritance:

class Animal:
    def __init__(self, name):
        self.name = name

    def make_sound(self):
        pass

class Dog(Animal):
    def make_sound(self):
        return "Woof!"

class Cat(Animal):
    def make_sound(self):
        return "Meow!"

# Example usage
dog = Dog("Buddy")
cat = Cat("Whiskers")

print(dog.name)         # Output: Buddy
print(dog.make_sound())  # Output: Woof!

print(cat.name)         # Output: Whiskers
print(cat.make_sound())  # Output: Meow!

In this example:

  • Animal is the base class with a common attribute (name) and a method (make_sound) that is meant to be overridden by the derived classes.

  • Dog and Cat are derived classes that inherit from the Animal class. They provide their own implementation of the make_sound method.

Method Overriding:

  • Method overriding allows a derived class to provide a specific implementation for a method that is already defined in the base class. In the example above, both Dog and Cat override the make_sound method.

Access to Base Class Methods:

  • The derived class can access the methods and attributes of the base class using the super() function.
class Shape:
    def __init__(self, color):
        self.color = color

    def display_color(self):
        print(f"Color: {self.color}")

class Square(Shape):
    def __init__(self, color, side_length):
        super().__init__(color)
        self.side_length = side_length

    def display_info(self):
        self.display_color()
        print(f"Side Length: {self.side_length}")

# Example usage
square = Square("Red", 5)
square.display_info()

In this example, the Square class calls the __init__ method of the base class Shape using super().__init__(color) to initialize the color attribute.

Multiple Inheritance:

  • Python supports multiple inheritance, allowing a class to inherit from multiple base classes.
class A:
    def method_a(self):
        print("Method A")

class B:
    def method_b(self):
        print("Method B")

class C(A, B):
    def method_c(self):
        print("Method C")

# Example usage
obj_c = C()
obj_c.method_a()  # Output: Method A
obj_c.method_b()  # Output: Method B
obj_c.method_c()  # Output: Method C

In this example, class C inherits from both classes A and B.

In summary, inheritance is a powerful concept in Python's object-oriented programming that allows for the creation of a hierarchy of classes, code reuse, and the organization of code in a modular and maintainable way.
