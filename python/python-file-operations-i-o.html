Python File Operations (I/O)

In Python, file operations are used to read from or write to files. Python provides built-in functions and methods for working with files. Here are some common file operations:

Reading from a File:

  1. open() function:

    • The open() function is used to open a file. It takes two arguments: the name of the file and the mode in which the file should be opened ('r' for reading, 'w' for writing, 'a' for appending, and more).
    # Open a file for reading
file = open('example.txt', 'r')

# Read the entire file
content = file.read()
print(content)

# Close the file
file.close()

  2. readline() method:

    • The readline() method reads a single line from the file.
    file = open('example.txt', 'r')
line = file.readline()
print(line)

# Close the file
file.close()

  3. readlines() method:

    • The readlines() method reads all lines from the file and returns them as a list.
    file = open('example.txt', 'r')
lines = file.readlines()
print(lines)

# Close the file
file.close()

Writing to a File:

  1. open() function with 'w' mode:

    • Open a file for writing. If the file already exists, its contents will be overwritten.
    # Open a file for writing
file = open('example.txt', 'w')

# Write content to the file
file.write('Hello, World!\n')
file.write('Python is awesome.')

# Close the file
file.close()

  2. writelines() method:

    • The writelines() method writes a list of lines to the file.
    lines = ['Line 1\n', 'Line 2\n', 'Line 3\n']

file = open('example.txt', 'w')
file.writelines(lines)
file.close()

Appending to a File:

  1. open() function with 'a' mode:

    • Open a file for appending. If the file does not exist, it will be created.
    # Open a file for appending
file = open('example.txt', 'a')

# Append content to the file
file.write('\nAppending new line.')

# Close the file
file.close()

Using with Statement:

It is recommended to use the with statement when working with files. This ensures that the file is properly closed after operations are performed.

with open('example.txt', 'r') as file:
    content = file.read()
    print(content)
# File is automatically closed after exiting the 'with' block

These examples cover the basics of file operations in Python. It's important to handle files carefully, close them after usage, and consider error handling for file-related operations.
