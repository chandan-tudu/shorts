Python Data Types

In Python, data types are classifications that specify which type of value a variable can hold. Python has several built-in data types, and here are some of the most common ones:

  1. Numeric Types:

    • int (integer): Integer values without a fractional component.

      x = 5

    • float (floating-point): Real numbers with a decimal point or in exponential form.

      y = 3.14

    • complex (complex numbers): Numbers with a real and imaginary part.

      z = 2 + 3j

  2. Sequence Types:

    • str (string): A sequence of characters.

      message = "Hello, Python!"

    • list (list): An ordered collection of items.

      my_list = [1, 2, 3, "apple", True]

    • tuple (tuple): An ordered, immutable collection of items.

      my_tuple = (1, 2, 3, "banana", False)

  3. Mapping Type:

    • dict (dictionary): A collection of key-value pairs. 
      
my_dict = {'name': 'John', 'age': 30, 'city': 'New York'}

  4. Set Types:

    • set (set): An unordered collection of unique items. 
      
my_set = {1, 2, 3, 4, 5}

  5. Boolean Type:

    • bool (boolean): Represents the truth values True or False. 
      
is_valid = True

  6. None Type:

    • NoneType (None): Represents the absence of a value. 
      
no_value = None

These are the primary built-in data types in Python. Additionally, Python allows you to create your own data types using classes (Object-Oriented Programming). Each data type has its own set of operations and methods associated with it. Understanding data types is fundamental to programming in Python and is crucial for writing correct and efficient code.
