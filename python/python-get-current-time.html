Python Get Current Time

In Python, you can use the datetime module to get the current time. Here's an example:

from datetime import datetime

# Get the current time
current_time = datetime.now().time()

print("Current time:", current_time)

In this example:

  • datetime.now() returns a datetime object representing the current date and time.

  • .time() extracts the time part from the datetime object, leaving you with a time object.

  • The print statement displays the current time.

If you want to format the time as a string, you can use the strftime method:

from datetime import datetime

# Get the current time and format it as a string
formatted_time = datetime.now().strftime("%H:%M:%S")

print("Formatted current time:", formatted_time)

In this example, the format string %H:%M:%S represents the hour, minute, and second components of the time in a 24-hour clock format.

Feel free to adjust the format string according to your specific needs.
