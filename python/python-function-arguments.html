Python Function Arguments

In Python, function arguments are values that are passed to a function when it is called. Functions can accept different types of arguments, and there are several ways to define and use them. Here are the main types of function arguments in Python:

  1. Positional Arguments:

    • These are the most common type of arguments. The values are passed based on the position of the parameters in the function definition.
    def add(x, y):
   return x + y

result = add(3, 5)
print(result)  # Output: 8

  2. Keyword Arguments:

    • Values are passed to a function using the parameter names. This allows you to specify which parameter each value corresponds to, regardless of their order.
    def greet(name, message):
   print(f"{message}, {name}!")

greet(message="Hello", name="John")

  3. Default Arguments:

    • You can provide default values for some or all of the function parameters. If a value is not provided when the function is called, the default value is used.
    def power(base, exponent=2):
   return base ** exponent

result = power(3)
print(result)  # Output: 9

  4. Variable-Length Argument Lists:

    • Functions can accept a variable number of arguments. There are two types:
      • *args: Allows a function to accept any number of positional arguments.
      • **kwargs: Allows a function to accept any number of keyword arguments.
    def sum_all(*args):
   return sum(args)

total = sum_all(1, 2, 3, 4)
print(total)  # Output: 10

    def print_info(**kwargs):
   for key, value in kwargs.items():
       print(f"{key}: {value}")

print_info(name="John", age=30, city="New York")

These different types of arguments provide flexibility and allow you to create functions that are versatile and can handle a variety of input scenarios.
