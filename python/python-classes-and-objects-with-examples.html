Python Classes And Objects With Examples

In Python, objects and classes are fundamental concepts of object-oriented programming (OOP). Here's an overview of how objects and classes work in Python:

Classes:

A class is a blueprint for creating objects. It defines a set of attributes (data) and methods (functions) that the objects of the class will have.

class Car:
    def __init__(self, make, model):
        self.make = make
        self.model = model

    def display_info(self):
        print(f"{self.make} {self.model}")

In this example, Car is a class with attributes make and model and a method display_info.

Objects:

An object is an instance of a class. It represents a specific instantiation of the class with its own unique data.

my_car = Car("Toyota", "Camry")

Here, my_car is an object of the Car class.

Instantiation:

The process of creating an object from a class is called instantiation. It involves calling the class as if it were a function, passing any necessary arguments.

another_car = Car("Honda", "Accord")

Here, another_car is another object created by instantiating the Car class.

Attributes and Methods:

Attributes are variables that store data, and methods are functions that perform actions. They are accessed using dot notation.

print(my_car.make)         # Output: Toyota
print(my_car.model)        # Output: Camry
my_car.display_info()      # Output: Toyota Camry

Constructor (__init__ method):

The __init__ method is a special method in Python classes that is called when an object is created. It initializes the object's attributes.

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

student = Student("Alice", 20)
print(student.name)  # Output: Alice
print(student.age)   # Output: 20

Class and Instance Variables:

Class variables are shared among all instances of a class, while instance variables are unique to each instance.

class BankAccount:
    interest_rate = 0.05  # Class variable

    def __init__(self, balance):
        self.balance = balance  # Instance variable

account1 = BankAccount(1000)
account2 = BankAccount(2000)

print(account1.balance)       # Output: 1000
print(account2.balance)       # Output: 2000
print(BankAccount.interest_rate)  # Output: 0.05

Class Methods and Static Methods:

Class methods are methods that are bound to the class and not the instance of the class, while static methods are independent of the class and the instance.

class MathOperations:
    @classmethod
    def add(cls, a, b):
        return a + b

    @staticmethod
    def multiply(a, b):
        return a * b

result1 = MathOperations.add(3, 4)         # Class method
result2 = MathOperations.multiply(3, 4)    # Static method

print(result1)  # Output: 7
print(result2)  # Output: 12

These are the basics of objects and classes in Python. Understanding these concepts is crucial for building modular, reusable, and maintainable code using object-oriented principles.
