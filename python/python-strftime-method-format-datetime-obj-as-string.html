Python strftime() Method: Format Datetime Obj As String

The strftime() method in Python's datetime module is used to format a datetime object as a string. It stands for "string format time" and allows you to represent a datetime object in a human-readable string format.

This method takes a format string as an argument, where various format codes represent different components of the date and time.

Here are some common format codes used with the strftime() method:

  • %Y: Year with century as a decimal number (e.g., 2022).
  • %m: Month as a zero-padded decimal number (01, 02, ..., 12).
  • %d: Day of the month as a zero-padded decimal number (01, 02, ..., 31).
  • %H: Hour (00, 01, ..., 23).
  • %M: Minute (00, 01, ..., 59).
  • %S: Second (00, 01, ..., 59).
  • %a: Abbreviated weekday name (Sun, Mon, ..., Sat).
  • %A: Full weekday name (Sunday, Monday, ..., Saturday).
  • %b: Abbreviated month name (Jan, Feb, ..., Dec).
  • %B: Full month name (January, February, ..., December).
  • %c: Localeâ€™s appropriate date and time representation.
  • %p: AM or PM.
  • %Z: Time zone name.

Here's an example of using strftime() to format a datetime object:

from datetime import datetime

# Creating a datetime object
now = datetime.now()

# Formatting the datetime object as a string
formatted_date = now.strftime("%Y-%m-%d %H:%M:%S")
print("Formatted date:", formatted_date)

This would output the current date and time in the specified format, like "2022-12-31 18:30:00".

You can customize the format string to represent the datetime object in various ways based on your specific requirements. Experimenting with different format codes will allow you to achieve the desired output.

# Example with additional format codes
custom_format = now.strftime("Today is %A, %B %d, %Y. The time is %I:%M %p.")
print(custom_format)

This might output something like "Today is Saturday, December 31, 2022. The time is 06:30 PM."
