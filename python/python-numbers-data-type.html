Python Numbers (Data Type)

In Python, numbers are a built-in data type used to represent numeric values. There are different types of numbers in Python, including integers, floating-point numbers, and complex numbers.

  1. Integers (int):

    • Integers represent whole numbers without any decimal point. They can be positive or negative.
    x = 5
y = -10
z = 0

  2. Floating-Point Numbers (float):

    • Floating-point numbers represent numbers with decimal points or numbers expressed in scientific notation.
    a = 3.14
b = -2.5
c = 2e3  # 2 times 10 to the power of 3 (2000.0)

  3. Complex Numbers (complex):

    • Complex numbers are represented as real + imagj, where real is the real part and imag is the imaginary part.
    complex_num = 2 + 3j

    The j or J suffix denotes the imaginary part.

Numeric operations can be performed on these types, and Python provides various built-in functions and modules for working with numbers. Here are some examples of numeric operations:

# Basic arithmetic operations
a = 5
b = 2

addition = a + b
subtraction = a - b
multiplication = a * b
division = a / b  # Result will be a floating-point number

# Exponentiation
exponential = a ** b  # a raised to the power of b

# Floor division (returns the largest integer less than or equal to the result)
floor_division = a // b

# Modulus (returns the remainder of the division)
remainder = a % b

# Absolute value
absolute_value = abs(-7)

# Round a floating-point number
rounded_number = round(3.14159, 2)  # Round to 2 decimal places

These examples demonstrate some of the basic numeric operations available in Python. Python's standard library also includes the math module, which provides additional mathematical functions and constants.
