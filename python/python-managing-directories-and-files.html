Python Managing Directories And Files

In Python, the os and shutil modules provide functions for managing directories and files. Here are some common operations related to directory and file management:

Directory Operations:

  1. Create a Directory:

    • Use the os.mkdir() function to create a new directory.
    import os

os.mkdir("my_directory")

  2. Create Directories Recursively:

    • Use the os.makedirs() function to create directories recursively, including any necessary parent directories.
    import os

os.makedirs("parent/child/grandchild")

  3. List Contents of a Directory:

    • Use the os.listdir() function to get a list of files and directories in a given directory.
    import os

contents = os.listdir("my_directory")
print(contents)

  4. Check if a Directory Exists:

    • Use the os.path.exists() function to check if a directory exists.
    import os

if os.path.exists("my_directory"):
   print("The directory exists.")
else:
   print("The directory does not exist.")

  5. Remove a Directory:

    • Use the os.rmdir() function to remove an empty directory.
    import os

os.rmdir("my_directory")

  6. Remove a Directory and its Contents Recursively:

    • Use the shutil.rmtree() function to remove a directory and its contents, including all subdirectories.
    import shutil

shutil.rmtree("parent")

File Operations:

  1. Check if a File Exists:

    • Use the os.path.isfile() function to check if a file exists.
    import os

if os.path.isfile("my_file.txt"):
   print("The file exists.")
else:
   print("The file does not exist.")

  2. Copy a File:

    • Use the shutil.copy() function to copy a file.
    import shutil

shutil.copy("source_file.txt", "destination_file.txt")

  3. Move/Rename a File:

    • Use the shutil.move() function to move or rename a file.
    import shutil

shutil.move("old_name.txt", "new_name.txt")

  4. Remove a File:

    • Use the os.remove() function to remove (delete) a file.
    import os

os.remove("file_to_delete.txt")

These examples cover some basic operations for managing directories and files in Python. Always be careful when performing file and directory operations, especially when deleting or overwriting files, to avoid accidental data loss.
