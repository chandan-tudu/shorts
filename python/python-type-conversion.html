Python Type Conversion

In Python, type conversion (or type casting) refers to the process of converting a variable from one data type to another.

Python provides built-in functions for performing type conversion. Here are some common type conversion functions:

  1. int(x [,base]):

    • Converts x to an integer. The optional base parameter specifies the base (default is 10).
    float_number = 3.14
integer_number = int(float_number)

  2. float(x):

    • Converts x to a floating-point number.
    integer_number = 42
float_number = float(integer_number)

  3. str(x):

    • Converts x to a string.
    number = 42
string_number = str(number)

  4. bool(x):

    • Converts x to a Boolean value.
    number = 42
boolean_value = bool(number)

  5. list(x):

    • Converts x to a list.
    string = "hello"
list_of_characters = list(string)

  6. tuple(x):

    • Converts x to a tuple.
    list_of_numbers = [1, 2, 3]
tuple_of_numbers = tuple(list_of_numbers)

  7. set(x):

    • Converts x to a set.
    list_of_numbers = [1, 2, 3, 3]
set_of_numbers = set(list_of_numbers)

  8. dict(x):

    • Converts x to a dictionary. For every pair of items in x, the first item becomes the key and the second item becomes the value.
    list_of_pairs = [('a', 1), ('b', 2), ('c', 3)]
dictionary = dict(list_of_pairs)

  9. chr(x):

    • Converts an integer x to a character.
    ascii_value = 65
character = chr(ascii_value)

  10. ord(x):

    • Converts a character x to its ASCII integer value.
    character = 'A'
ascii_value = ord(character)

  11. hex(x):

    • Converts an integer x to a lowercase hexadecimal string.
    decimal_number = 255
hex_string = hex(decimal_number)

  12. bin(x):

    • Converts an integer x to a binary string.
    decimal_number = 7
binary_string = bin(decimal_number)

  13. oct(x):

    • Converts an integer x to an octal string.
    decimal_number = 63
octal_string = oct(decimal_number)

These functions enable you to convert values between different data types, which can be useful in various scenarios, such as user input handling, data manipulation, and more.
