Python "math" Module

In Python, the math module provides a set of mathematical functions for operations involving numbers. To use the math module, you need to import it first. Here are some common functions and constants provided by the math module:

  1. Importing the math module:

    import math

  2. Mathematical Constants:

    • math.pi: Returns the mathematical constant Ï€ (pi).

      print(math.pi)  # Output: 3.141592653589793

    • math.e: Returns the mathematical constant e.

      print(math.e)   # Output: 2.718281828459045

  3. Basic Mathematical Functions:

    • math.sqrt(x): Returns the square root of x.

      result = math.sqrt(25)
print(result)  # Output: 5.0

    • math.pow(x, y): Returns x raised to the power y.

      result = math.pow(2, 3)
print(result)  # Output: 8.0

    • math.exp(x): Returns e raised to the power x.

      result = math.exp(2)
print(result)  # Output: 7.3890560989306495

    • math.log(x): Returns the natural logarithm of x (base e).

      result = math.log(10)
print(result)  # Output: 2.302585092994046

    • math.log10(x): Returns the base-10 logarithm of x.

      result = math.log10(100)
print(result)  # Output: 2.0

  4. Trigonometric Functions:

    • math.sin(x), math.cos(x), math.tan(x): Return the sine, cosine, and tangent of x (in radians).

      angle = math.pi / 4
print(math.sin(angle))  # Output: 0.7071067811865475 (approximately)

    • math.degrees(x), math.radians(x): Convert between degrees and radians.

      radians_value = math.radians(90)
print(math.degrees(radians_value))  # Output: 90.0

  5. Other Functions:

    • math.floor(x), math.ceil(x): Return the floor and ceiling of x as a float.

      print(math.floor(4.8))  # Output: 4.0
print(math.ceil(4.1))   # Output: 5.0

    • math.factorial(x): Return x factorial (x!).

      result = math.factorial(5)
print(result)  # Output: 120

    • math.fabs(x): Return the absolute value of x as a float.

      result = math.fabs(-7.5)
print(result)  # Output: 7.5

These are just a few examples of the functions provided by the math module. For more details, you can refer to the official documentation.
