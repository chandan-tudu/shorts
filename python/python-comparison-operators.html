Python Comparison Operators

In Python, comparison operators are used to compare two values and return a Boolean result (True or False). Here are the commonly used comparison operators:

  1. Equal ==:

    • Returns True if the values on both sides are equal.
    result = (5 == 5)  # True

  2. Not Equal !=:

    • Returns True if the values on both sides are not equal.
    result = (7 != 3)  # True

  3. Greater Than >:

    • Returns True if the value on the left is greater than the value on the right.
    result = (8 > 5)  # True

  4. Less Than <:

    • Returns True if the value on the left is less than the value on the right.
    result = (4 < 2)  # False

  5. Greater Than or Equal To >=:

    • Returns True if the value on the left is greater than or equal to the value on the right.
    result = (10 >= 10)  # True

  6. Less Than or Equal To <=:

    • Returns True if the value on the left is less than or equal to the value on the right.
    result = (3 <= 1)  # False

These operators are often used in conditional statements, loops, and other situations where you need to make decisions based on the relationships between values.

Example:

x = 10
y = 20

# Comparison operators in conditional statements
if x == y:
    print("x is equal to y")
elif x < y:
    print("x is less than y")
else:
    print("x is greater than y")

In this example, the program uses comparison operators to check the relationship between x and y and prints the appropriate message based on the result.
