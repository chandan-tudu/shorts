Python strptime() Method: Parse String DateTime

The strptime() method in Python's datetime module is used to parse a string representing a date and time into a datetime object. It stands for "string parse time" and is the counterpart to the strftime() method, which formats a datetime object as a string.

The strptime() method takes two arguments: the string to be parsed and the format string that specifies the expected format of the input string. The format string uses format codes to represent various components of the date and time.

Here's a basic example:

from datetime import datetime

# Parsing a string into a datetime object
date_string = "2022-12-31 18:30:00"
parsed_date = datetime.strptime(date_string, "%Y-%m-%d %H:%M:%S")

print("Parsed date:", parsed_date)

In this example, the date_string is parsed using the format string "%Y-%m-%d %H:%M:%S", which corresponds to the "Year-Month-Day Hour:Minute:Second" format. The resulting datetime object is printed.

Common format codes used with strptime():

  • %Y: Year with century as a decimal number.
  • %m: Month as a zero-padded decimal number.
  • %d: Day of the month as a zero-padded decimal number.
  • %H: Hour (00, 01, ..., 23).
  • %M: Minute (00, 01, ..., 59).
  • %S: Second (00, 01, ..., 59).

Here's another example with a different date format:

from datetime import datetime

# Parsing a string with a different date format
date_string = "31-12-2022 18:30:00"
parsed_date = datetime.strptime(date_string, "%d-%m-%Y %H:%M:%S")

print("Parsed date:", parsed_date)

It's important to ensure that the format string matches the structure of the input string; otherwise, a ValueError will be raised.
