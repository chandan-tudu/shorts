Python Get User Input

In Python, you can get user input using the input() function. The input() function takes a string as an argument, which is typically a prompt or message displayed to the user. Here's a simple example:

# Get user input
user_name = input("Enter your name: ")

# Display the user input
print("Hello, " + user_name + "!")

In this example, the input() function prompts the user to enter their name. The entered text is then assigned to the variable user_name, and it is displayed in a greeting message.

It's important to note that the input() function always returns a string. If you need to convert the input to a different type (e.g., integer or float), you can use the appropriate conversion function like int() or float():

# Get user input as an integer
user_age = int(input("Enter your age: "))

# Display the user input
print("You are " + str(user_age) + " years old.")

In this case, int(input(...)) is used to convert the user input to an integer. If the user enters something that cannot be converted to an integer, a ValueError will occur, so it's a good practice to handle such cases if necessary.
