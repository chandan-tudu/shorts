Python While Loop

In Python, the while loop is used to repeatedly execute a block of code as long as a specified condition is true. The basic syntax of a while loop is as follows:

while condition:
    # Code to be executed while the condition is true

Here's a simple example that counts from 1 to 5 using a while loop:

count = 1

while count <= 5:
    print(count)
    count += 1

In this example, the loop will continue to execute as long as the condition count <= 5 is true. Inside the loop, the current value of count is printed, and then count is incremented by 1. The loop stops when the condition becomes false.

Output:

1
2
3
4
5

It's important to ensure that the loop's condition will eventually become false; otherwise, the loop will continue indefinitely, resulting in an infinite loop.

Here's an example of a while loop that asks the user for input until a specific condition is met:

user_input = ""

while user_input.lower() != "quit":
    user_input = input("Enter a word (or 'quit' to exit): ")
    print("You entered:", user_input)

In this case, the loop will keep asking the user for input until they enter the word "quit" (case-insensitive). The loop condition checks whether user_input is not equal to "quit."
