Python Set Methods With Examples

In Python, sets come with a variety of built-in methods that allow you to perform operations and manipulations on sets. Here are some common set methods:

  1. add(element) method:

    • Adds a specified element to the set.
    my_set = {1, 2, 3}
my_set.add(4)
print(my_set)  # Output: {1, 2, 3, 4}

  2. remove(element) method:

    • Removes the specified element from the set. Raises a KeyError if the element is not present.
    my_set = {1, 2, 3, 4}
my_set.remove(3)
print(my_set)  # Output: {1, 2, 4}

  3. discard(element) method:

    • Removes the specified element from the set if it is present. Does not raise an error if the element is not found.
    my_set = {1, 2, 3, 4}
my_set.discard(3)
print(my_set)  # Output: {1, 2, 4}

  4. pop() method:

    • Removes and returns an arbitrary element from the set. Raises a KeyError if the set is empty.
    my_set = {1, 2, 3, 4}
popped_element = my_set.pop()
print(popped_element)
print(my_set)  # Output: An arbitrary element and the updated set

  5. clear() method:

    • Removes all elements from the set.
    my_set = {1, 2, 3, 4}
my_set.clear()
print(my_set)  # Output: set()

  6. copy() method:

    • Returns a shallow copy of the set.
    my_set = {1, 2, 3, 4}
copied_set = my_set.copy()
print(copied_set)

  7. union() method or | operator:

    • Returns a new set containing all unique elements from both sets.
    set1 = {1, 2, 3}
set2 = {3, 4, 5}
union_set = set1.union(set2)
# Alternatively: union_set = set1 | set2
print(union_set)  # Output: {1, 2, 3, 4, 5}

  8. intersection() method or & operator:

    • Returns a new set containing common elements from both sets.
    set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}
intersection_set = set1.intersection(set2)
# Alternatively: intersection_set = set1 & set2
print(intersection_set)  # Output: {3, 4}

  9. difference() method or - operator:

    • Returns a new set containing elements that are present in the first set but not in the second set.
    set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}
difference_set = set1.difference(set2)
# Alternatively: difference_set = set1 - set2
print(difference_set)  # Output: {1, 2}

  10. symmetric_difference() method or ^ operator:

    • Returns a new set containing elements that are present in either set, but not both.
    set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}
symmetric_difference_set = set1.symmetric_difference(set2)
# Alternatively: symmetric_difference_set = set1 ^ set2
print(symmetric_difference_set)  # Output: {1, 2, 5, 6}

These are just a few examples of the methods available for sets in Python. Sets provide efficient ways to perform common set operations and are particularly useful when dealing with unique elements.
