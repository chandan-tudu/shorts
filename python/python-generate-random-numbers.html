Python Generate Random Numbers

In Python, you can generate random numbers using the random module. Here's a simple example:

import random

# Generate a random float between 0 and 1
random_float = random.random()
print("Random float between 0 and 1:", random_float)

# Generate a random integer within a specified range
random_integer = random.randint(1, 100)
print("Random integer between 1 and 100:", random_integer)

In this example:

  • random.random() generates a random float between 0 (inclusive) and 1 (exclusive).
  • random.randint(a, b) generates a random integer between a and b (both inclusive).

If you want to generate a random number with decimal places within a specific range, you can use random.uniform(a, b). Here's an example:

# Generate a random float within a specified range
random_float_range = random.uniform(2.5, 5.5)
print("Random float between 2.5 and 5.5:", random_float_range)

Remember to import the random module before using these functions.
