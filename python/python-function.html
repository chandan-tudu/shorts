Python Function

In Python, a function is a block of reusable code that performs a specific task. Functions provide a way to structure code, promote reusability, and make the code more modular. Here's a basic overview of defining and using functions in Python:

1. Defining a Function:

You can define a function using the def keyword, followed by the function name, a pair of parentheses (which may include parameters), and a colon. The function body is indented.

def greet(name):
    # This function greets the person passed in as a parameter.
    print("Hello, " + name + "!")

2. Calling a Function:

To execute a function, you simply call it by using its name followed by parentheses. If the function requires input, you provide the arguments inside the parentheses.

greet("Alice")

3. Function Parameters:

Functions can take parameters, which are variables that allow the function to receive inputs.

def add_numbers(a, b):
    # This function adds two numbers.
    result = a + b
    return result

4. Function Return:

Functions can return values using the return statement. The calling code can use the returned value.

sum_result = add_numbers(3, 4)
print("Sum:", sum_result)

6. Default Parameters:

You can provide default values for parameters. If a value is not passed, the default value is used.

def power(base, exponent=2):
    # This function calculates the power of a number.
    result = base ** exponent
    return result

7. Keyword Arguments:

You can use keyword arguments to pass values to a function based on the parameter names.

result = power(base=3, exponent=4)

8 Variable-Length Argument Lists:

A function can accept a variable number of arguments using *args and **kwargs.

def print_args(*args, **kwargs):
    # This function prints all positional and keyword arguments.
    print("Positional arguments:", args)
    print("Keyword arguments:", kwargs)

9. Lambda Functions (Anonymous Functions):

Lambda functions are small, anonymous functions defined using the lambda keyword.

square = lambda x: x**2
print(square(5))

10. Docstrings:

It's good practice to include docstrings (triple-quoted strings) to document the purpose and usage of your functions.

def multiply(a, b):
    """This function multiplies two numbers."""
    result = a * b
    return result

These are the basics of defining and using functions in Python. Functions play a crucial role in organizing and structuring code, improving code readability, and facilitating code reuse.
