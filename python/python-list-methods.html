Python List Methods

Python lists come with a variety of built-in methods that you can use to manipulate and work with lists. Here are some common list methods:

List Methods for Adding Elements:

  1. append(x): Adds an element x to the end of the list.

    my_list = [1, 2, 3]
my_list.append(4)
print(my_list)

  2. extend(iterable): Extends the list by appending elements from the iterable.

    my_list = [1, 2, 3]
my_list.extend([4, 5, 6])
print(my_list)

  3. insert(i, x): Inserts element x at index i in the list.

    my_list = [1, 2, 3]
my_list.insert(1, 4)
print(my_list)

List Methods for Removing Elements:

  1. remove(x): Removes the first occurrence of element x from the list.

    my_list = [1, 2, 3, 2, 4]
my_list.remove(2)
print(my_list)

  2. pop([i]): Removes and returns the element at index i. If i is not provided, it removes and returns the last element.

    my_list = [1, 2, 3]
popped_element = my_list.pop(1)
print("Popped element:", popped_element)
print("List after popping index 1:", my_list)

List Methods for Searching and Counting:

  1. index(x[, start[, end]]): Returns the index of the first occurrence of element x in the list. Raises a ValueError if x is not found.

    my_list = [1, 2, 3, 2, 4]
index_of_3 = my_list.index(3)
print("Index of 3:", index_of_3)

  2. count(x): Returns the number of occurrences of element x in the list.

    my_list = [1, 2, 3, 2, 4]
count_of_2 = my_list.count(2)
print("Count of 2:", count_of_2)

List Methods for Sorting and Reversing:

  1. sort(key=None, reverse=False): Sorts the elements of the list in ascending order. The key and reverse parameters allow customization.

    my_list = [4, 2, 1, 3]
my_list.sort()
print("Sorted list:", my_list)

  2. reverse(): Reverses the elements of the list in-place.

    my_list = [1, 2, 3, 4]
my_list.reverse()
print("Reversed list:", my_list)

These are just some of the commonly used methods for lists in Python. The official Python documentation provides a comprehensive list of list methods along with detailed explanations: Python List Methods.
