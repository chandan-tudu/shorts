How To Declare A Variable In Python

In Python, you can declare a variable by simply assigning a value to it. Python is dynamically typed, meaning you don't need to explicitly declare the type of a variable; the interpreter infers the type based on the value assigned. Here's an example:

# Variable declaration and assignment
my_variable = 42

# Print the value of the variable
print(my_variable)

In this example, my_variable is declared and assigned the value 42. Python recognizes that my_variable is an integer because of the value assigned to it.

You can also declare multiple variables in a single line:

# Multiple variable declaration and assignment
x, y, z = 10, 20, 30

# Print the values of the variables
print(x, y, z)

In this case, x, y, and z are declared and assigned the values 10, 20, and 30 respectively.

It's important to note that in Python, variables are more like labels or references to objects, rather than containers that hold values. This has implications when dealing with mutable objects like lists. If you modify a mutable object through one variable, the change will be reflected in other variables that reference the same object.
