Python break And continue Statements

In Python, the break and continue statements are used to control the flow of loops.

  1. break Statement:

    • The break statement is used to exit a loop prematurely. It is typically used within a loop's body to terminate the loop based on a certain condition.
    # Example using break in a while loop
count = 1

while True:
   print(count)
   count += 1
   if count > 5:
       break

    In this example, the break statement is used to exit the while loop when the count exceeds 5.

  2. continue Statement:

    • The continue statement is used to skip the rest of the code inside a loop for the current iteration and move to the next iteration.
    # Example using continue in a for loop
for i in range(1, 6):
   if i == 3:
       continue
   print(i)

    In this example, the continue statement skips the print statement when i is equal to 3, and the loop proceeds to the next iteration.

Here's a combined example using both break and continue:

# Example using break and continue in a while loop
count = 1

while count <= 5:
    if count == 3:
        count += 1
        continue
    print(count)
    count += 1
    if count > 4:
        break

Output:

1
2
4

In this example, the continue statement skips the print statement when count is equal to 3, and the break statement exits the loop when count exceeds 4.

These statements provide flexibility in controlling the flow of execution within loops based on certain conditions.
