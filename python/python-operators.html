Python Operators

In Python, operators are special symbols or keywords that perform operations on operands. Operands are the values or variables that the operator acts upon.

Python supports a variety of operators, and they can be classified into different categories. Here's an overview of some common operators in Python:

1. Arithmetic Operators:

These operators perform mathematical operations on numeric values.

  • Addition +:

    result = 3 + 5  # 8

  • Subtraction -:

    result = 7 - 4  # 3

  • Multiplication *:

    result = 2 * 6  # 12

  • Division /:

    result = 10 / 2  # 5.0 (float result)

  • Floor Division //:

    result = 10 // 3  # 3 (integer result, discarding remainder)

  • Modulus %:

    result = 10 % 3  # 1 (remainder after division)

  • **Exponentiation **:**

    result = 2 ** 3  # 8 (2 raised to the power of 3)

2. Comparison Operators:

These operators compare two values and return a Boolean result (True or False).

  • Equal ==:

    result = (5 == 5)  # True

  • Not Equal !=:

    result = (7 != 3)  # True

  • Greater Than >:

    result = (8 > 5)  # True

  • Less Than <:

    result = (4 < 2)  # False

  • Greater Than or Equal To >=:

    result = (10 >= 10)  # True

  • Less Than or Equal To <=:

    result = (3 <= 1)  # False

3. Logical Operators:

These operators perform logical operations on Boolean values.

  • Logical AND and:

    result = (True and False)  # False

  • Logical OR or:

    result = (True or False)  # True

  • Logical NOT not:

    result = not True  # False

4. Assignment Operators:

These operators assign values to variables.

  • Assignment =:

    x = 5

  • Addition Assignment +=:

    x += 3  # Equivalent to x = x + 3

  • Subtraction Assignment -=:

    y -= 2  # Equivalent to y = y - 2

5. Other Operators:

  • Membership Operators (in, not in):

    result = (2 in [1, 2, 3])  # True

  • Identity Operators (is, is not):

    result = (x is y)  # False (checks if x and y refer to the same object)

  • Bitwise Operators (&, |, ^, ~, <<, >>):

    result = (5 & 3)  # Bitwise AND (result: 1)

These are just a few examples of the many operators available in Python. Understanding and using operators effectively is crucial for writing expressive and efficient code.
