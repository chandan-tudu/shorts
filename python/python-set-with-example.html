Python Set With Example

In Python, a set is an unordered collection of unique elements. Sets are defined by enclosing a comma-separated sequence of values in curly braces {}. Here are some examples:

# Creating a set
my_set = {1, 2, 3, 4, 5}

# Sets can also be created using the set() constructor
another_set = set([3, 4, 5, 6, 7])

Key characteristics of sets in Python:

  1. Unordered:

    • Sets do not maintain the order of elements as they are inserted.

  2. Unique Elements:

    • Sets do not allow duplicate elements. If you try to add a duplicate element, it won't be added.
    my_set = {1, 2, 3, 2, 4, 5}
print(my_set)  # Output: {1, 2, 3, 4, 5}

  3. Mutable:

    • Sets are mutable, meaning you can add and remove elements after the set is created.
    my_set.add(6)    # Add an element
my_set.remove(3) # Remove an element

  4. Common Set Operations:

    • Sets support various operations such as union (|), intersection (&), difference (-), and symmetric difference (^).
    set1 = {1, 2, 3, 4, 5}
set2 = {4, 5, 6, 7, 8}

union_set = set1 | set2            # Union
intersection_set = set1 & set2     # Intersection
difference_set = set1 - set2       # Difference
symmetric_difference_set = set1 ^ set2  # Symmetric Difference

  5. Set Methods:

    • Sets have several built-in methods for common set operations, including add(), remove(), discard(), clear(), copy(), and more.
    my_set = {1, 2, 3, 4, 5}

my_set.add(6)          # Add an element
my_set.remove(3)       # Remove an element
my_set.discard(2)      # Discard an element (if it exists)
my_set.clear()         # Remove all elements

Sets are often used in scenarios where you need to perform mathematical set operations or ensure the uniqueness of elements in a collection. They provide a convenient and efficient way to work with unordered collections of unique elements.
