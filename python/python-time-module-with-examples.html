Python "time" Module With Examples

The time module in Python provides various functions for working with time-related tasks, including measuring time intervals, formatting time, and working with the system clock.

Here are some common functions provided by the time module:

1. time(): Get Current Time

The time() function returns the current time in seconds since the epoch (January 1, 1970, 00:00:00 UTC).

import time

current_time = time.time()
print("Current time:", current_time)

2. sleep(): Pause Execution

The sleep() function suspends the execution of the program for a specified number of seconds.

import time

print("Start sleeping")
time.sleep(3)
print("End sleeping")

3. localtime(): Convert Time to Local Time

The localtime() function converts a time in seconds since the epoch to a struct_time in local time.

import time

current_time = time.time()
local_time = time.localtime(current_time)
print("Local time:", local_time)

4. asctime(): Format Time as String

The asctime() function formats a struct_time as a string in the format "Day Month DayOfMonth Hour:Minute:Second Year".

import time

current_time = time.time()
local_time = time.localtime(current_time)
formatted_time = time.asctime(local_time)
print("Formatted time:", formatted_time)

5. strftime(): Format Time with Custom Format

The strftime() function formats a struct_time using a specified format.

import time

current_time = time.time()
local_time = time.localtime(current_time)
formatted_time = time.strftime("%Y-%m-%d %H:%M:%S", local_time)
print("Formatted time:", formatted_time)

6. gmtime(): Convert Time to UTC Time

The gmtime() function converts a time in seconds since the epoch to a struct_time in UTC.

import time

current_time = time.time()
utc_time = time.gmtime(current_time)
print("UTC time:", utc_time)

7. clock(): CPU Time

The clock() function returns the current CPU time (in seconds) as a floating-point number.

import time

cpu_time = time.clock()
print("CPU time:", cpu_time)

Note: The clock() function has been deprecated in Python 3.3 and removed in Python 3.8. It is recommended to use time.process_time() instead for measuring CPU time.

These are just a few examples of the functionalities provided by the time module. For more detailed information, refer to the official documentation: Python time module documentation.
