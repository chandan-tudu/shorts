Python Logical Operators

Logical operators in Python are used to perform logical operations on Boolean values. These operators evaluate expressions and return a Boolean result (True or False). The common logical operators in Python are:

  1. Logical AND (and):

    • Returns True if both operands are true.
    result = (True and False)  # False

  2. Logical OR (or):

    • Returns True if at least one operand is true.
    result = (True or False)  # True

  3. Logical NOT (not):

    • Returns True if the operand is false, and False if the operand is true.
    result = not True  # False

Example:

x = 5
y = 10

# Logical operators in conditional statements
if x > 0 and y < 20:
    print("Both conditions are true")

if x > 0 or y > 20:
    print("At least one condition is true")

if not x < 0:
    print("The negation of the condition is true")

In this example, logical operators are used in conditional statements to check the combined truth values of different conditions. These operators are fundamental for creating more complex conditions and decision-making in your Python programs.
