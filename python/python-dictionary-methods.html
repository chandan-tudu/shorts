Python Dictionary Methods

Dictionaries in Python come with a variety of built-in methods that allow you to perform various operations. Here are some common dictionary methods:

Dictionary Methods for Accessing Elements:

  1. get(key, default): Returns the value for the specified key. If the key is not found, it returns the default value (or None if not specified).

    my_dict = {'name': 'John', 'age': 30}
value = my_dict.get('name', 'Default Name')
print(value)

  2. keys(): Returns a view of all keys in the dictionary.

    my_dict = {'name': 'John', 'age': 30}
all_keys = my_dict.keys()
print(all_keys)

  3. values(): Returns a view of all values in the dictionary.

    my_dict = {'name': 'John', 'age': 30}
all_values = my_dict.values()
print(all_values)

  4. items(): Returns a view of all key-value pairs in the dictionary as tuples.

    my_dict = {'name': 'John', 'age': 30}
all_items = my_dict.items()
print(all_items)

Dictionary Methods for Modifying Dictionaries:

  1. update(dict): Updates the dictionary with elements from another dictionary or from an iterable of key-value pairs.

    my_dict = {'name': 'John', 'age': 30}
my_dict.update({'city': 'New York'})
print(my_dict)

  2. pop(key, default): Removes the item with the specified key and returns its value. If the key is not found, it returns the default value (or raises a KeyError if not specified).

    my_dict = {'name': 'John', 'age': 30}
popped_value = my_dict.pop('age', None)
print(popped_value)

  3. popitem(): Removes and returns the last inserted key-value pair as a tuple.

    my_dict = {'name': 'John', 'age': 30}
popped_item = my_dict.popitem()
print(popped_item)

Dictionary Methods for Creating a Copy:

  1. copy(): Returns a shallow copy of the dictionary.

    my_dict = {'name': 'John', 'age': 30}
copy_dict = my_dict.copy()
print(copy_dict)

Dictionary Methods for Clearing:

  1. clear(): Removes all items from the dictionary.

    my_dict = {'name': 'John', 'age': 30}
my_dict.clear()
print(my_dict)

These are just a few examples of the methods available for dictionaries in Python. Depending on your specific use case, you may find other methods in the official Python documentation that suit your needs.
