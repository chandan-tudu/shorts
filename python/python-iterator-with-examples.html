Python Iterator With Examples

In Python, an iterator is an object that implements the iterator protocol, which consists of the __iter__ method and the __next__ method. Iterators are used to represent a stream of data and enable iteration over that data. Common examples of iterators in Python include lists, tuples, and dictionaries.

Here's a simple example of creating a custom iterator class:

class MyIterator:
    def __init__(self, data):
        self.data = data
        self.index = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.index < len(self.data):
            result = self.data[self.index]
            self.index += 1
            return result
        else:
            raise StopIteration

# Example usage
my_list = [1, 2, 3, 4, 5]
my_iterator = MyIterator(my_list)

for item in my_iterator:
    print(item)

In this example:

  • The MyIterator class has the __init__ method to initialize the iterator with some data and the __next__ method to provide the next element in the iteration.

  • The __iter__ method is implemented to return the iterator object itself.

  • When the iterator reaches the end of the data, it raises the StopIteration exception to signal the end of the iteration.

  • The for loop is used to iterate over the elements of the iterator.

Python also provides a convenient way to create iterators using generator functions and the iter function. Here's an example using a generator:

def my_generator(data):
    for item in data:
        yield item

# Example usage
my_list = [1, 2, 3, 4, 5]
my_iterator = iter(my_generator(my_list))

for item in my_iterator:
    print(item)

In this example, the my_generator function is a generator function that yields items from the input data. The iter function is then used to obtain an iterator from the generator.

Additionally, Python provides built-in functions like iter and next that work with any iterable object, making it easy to work with iterators in a general way:

my_list = [1, 2, 3, 4, 5]
my_iterator = iter(my_list)

while True:
    try:
        item = next(my_iterator)
        print(item)
    except StopIteration:
        break

This demonstrates the use of the iter function to obtain an iterator from a list and the next function to retrieve the next item in the iteration until a StopIteration exception is raised.

Iterators are an essential part of Python's iterable protocol, providing a uniform way to iterate over different types of data structures.
