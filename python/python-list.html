Python List

In Python, a list is a mutable, ordered collection of elements. Lists are defined using square brackets [] and can contain elements of different data types. Here's a basic example:

# Creating a list
my_list = [1, 2, 3, 'apple', 'banana', True]

# Accessing elements by index
print("Element at index 0:", my_list[0])
print("Element at index 3:", my_list[3])

Lists in Python are zero-indexed, meaning the first element is at index 0.

You can perform various operations on lists:

List Slicing:

# Slicing a list
sublist = my_list[1:4]  # Elements from index 1 to 3 (excluding 4)
print("Sliced list:", sublist)

Modifying Lists:

# Modifying elements
my_list[2] = 'orange'
print("Modified list:", my_list)

# Appending an element
my_list.append(4)
print("List after appending 4:", my_list)

# Extending the list with another list
my_list.extend([5, 6, 7])
print("Extended list:", my_list)

# Inserting an element at a specific index
my_list.insert(2, 'grape')
print("List after inserting 'grape' at index 2:", my_list)

# Removing an element by value
my_list.remove('banana')
print("List after removing 'banana':", my_list)

# Removing an element by index
popped_value = my_list.pop(3)
print("Popped value at index 3:", popped_value)
print("List after popping index 3:", my_list)

List Methods:

# Finding the index of an element
index_of_apple = my_list.index('apple')
print("Index of 'apple':", index_of_apple)

# Counting occurrences of an element
count_of_3 = my_list.count(3)
print("Count of 3:", count_of_3)

# Reversing the list
my_list.reverse()
print("Reversed list:", my_list)

# Sorting the list
my_list.sort()
print("Sorted list:", my_list)

List Comprehensions:

List comprehensions provide a concise way to create lists:

# Using list comprehension to create a new list
squared_numbers = [x**2 for x in range(5)]
print("Squared numbers:", squared_numbers)

Lists are versatile and widely used in Python for storing and manipulating collections of items. They are an essential part of the language and can be used in various scenarios.
