Python Program To Catch Multiple Exceptions In One Line

In Python, you can catch multiple exceptions in one line by using parentheses to create a tuple of exception types. Here's an example:

try:
    # Code that may raise exceptions
    x = 10 / 0
except (ZeroDivisionError, NameError, TypeError) as e:
    print(f"Caught an exception: {e}")

In this example, the try block contains code that might raise exceptions, such as a ZeroDivisionError, NameError, or TypeError. The except block catches any of these specified exceptions and prints a message.

You can customize the tuple of exception types based on the specific exceptions you want to catch. If any of the specified exceptions occur, the corresponding except block will be executed.

This approach is concise and allows you to handle multiple exception types in a single except block.
