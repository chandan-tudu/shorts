Python Program To Find The Factors Of A Number

Here's a Python program to find the factors of a given number:

def find_factors(number):
    factors = []
    for i in range(1, number + 1):
        if number % i == 0:
            factors.append(i)
    return factors

# Get input from the user
num = int(input("Enter a number to find its factors: "))

# Find and display the factors
result = find_factors(num)
print(f"The factors of {num} are: {result}")

Example:

Enter a number to find its factors: 24
The factors of 24 are: [1, 2, 3, 4, 6, 8, 12, 24]

In this program, the find_factors function takes a number as input and iterates through the range from 1 to the number, checking if each number is a factor (divisor) of the given number. If it is, the factor is added to the list. The user is prompted to enter a number, and the program then finds and displays its factors.
