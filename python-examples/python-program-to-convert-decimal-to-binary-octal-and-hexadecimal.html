Python Program To Convert Decimal To Binary, Octal And Hexadecimal

Here's a Python program that converts a decimal number to its binary, octal, and hexadecimal representations:

def decimal_to_binary_octal_hexadecimal(decimal_number):
    binary_representation = bin(decimal_number)[2:]
    octal_representation = oct(decimal_number)[2:]
    hexadecimal_representation = hex(decimal_number)[2:]

    return binary_representation, octal_representation, hexadecimal_representation

# Get input from the user
decimal_num = int(input("Enter a decimal number: "))

# Convert and display the representations
binary, octal, hexadecimal = decimal_to_binary_octal_hexadecimal(decimal_num)
print(f"Binary: {binary}")
print(f"Octal: {octal}")
print(f"Hexadecimal: {hexadecimal}")

Example:

Enter a decimal number: 255
Binary: 11111111
Octal: 377
Hexadecimal: ff

In this program, the decimal_to_binary_octal_hexadecimal function takes a decimal number as input and uses Python's built-in functions bin, oct, and hex to convert it to binary, octal, and hexadecimal representations, respectively. The user is prompted to enter a decimal number, and the program then converts and displays its binary, octal, and hexadecimal representations.
