Python Program To Capitalize The First Character Of A String

You can capitalize the first character of a string in Python using the capitalize() method or by manually manipulating the string. Here's an example using both approaches:

1. Using capitalize() Method:

def capitalize_first_char(input_str):
    return input_str.capitalize()

# Example usage:
original_string = "hello world"

capitalized_result = capitalize_first_char(original_string)

print(f"Original String: {original_string}")
print(f"Capitalized String: {capitalized_result}")

Output:

Original String: hello world
Capitalized String: Hello world

2. Manually Capitalizing:

def manually_capitalize_first_char(input_str):
    if input_str:
        return input_str[0].upper() + input_str[1:]
    else:
        return input_str

# Example usage:
original_string = "hello world"

manually_capitalized_result = manually_capitalize_first_char(original_string)

print(f"Original String: {original_string}")
print(f"Manually Capitalized String: {manually_capitalized_result}")

Output:

Original String: hello world
Manually Capitalized String: Hello world

In the second example, the manually_capitalize_first_char function manually capitalizes the first character by using string slicing and the upper() method.

Choose the method that best fits your requirements. Adjust the original_string variable in the examples to capitalize different strings.
