Python Program To Check If Two Strings Are Anagram

An anagram is a word or phrase formed by rearranging the letters of another. You can check if two strings are anagrams in Python by comparing the sorted versions of their characters. Here's an example:

def are_anagrams(str1, str2):
    # Remove spaces and convert to lowercase for case-insensitive comparison
    str1 = str1.replace(" ", "").lower()
    str2 = str2.replace(" ", "").lower()

    # Check if the sorted characters of both strings are the same
    return sorted(str1) == sorted(str2)

# Example usage:
string1 = "Listen"
string2 = "Silent"

if are_anagrams(string1, string2):
    print(f"{string1} and {string2} are anagrams.")
else:
    print(f"{string1} and {string2} are not anagrams.")

Output:

Listen and Silent are anagrams.

In this example, the are_anagrams function removes spaces and converts both strings to lowercase for case-insensitive comparison. It then checks if the sorted characters of both strings are the same.

Adjust the string1 and string2 variables in the example to check if different strings are anagrams.
