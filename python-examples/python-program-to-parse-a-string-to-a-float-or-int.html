Python Program To Parse A String To A Float Or Int

You can use the float() and int() functions to parse a string to a float or int in Python. Here's an example:

def parse_to_float_or_int(input_str):
    try:
        result_float = float(input_str)
        result_int = int(result_float)

        # Check if the parsed float is equal to the parsed int
        if result_float == result_int:
            return result_int
        else:
            return result_float
    except ValueError:
        print("Error: Unable to parse the string to a float or int.")
        return None

# Example usage:
input_string = "123.45"
parsed_value = parse_to_float_or_int(input_string)

if parsed_value is not None:
    print(f"Parsed value: {parsed_value}")

Output:

Parsed value: 123.45

In this example, the parse_to_float_or_int function attempts to parse the input string to a float and then to an int. If the float and int representations are equal, it returns the integer; otherwise, it returns the float. The ValueError exception is caught in case the string cannot be parsed to a float or int.

You can replace the input_string with the desired string you want to parse. Adjust the program accordingly if you expect a different format or if you want to handle additional cases.
