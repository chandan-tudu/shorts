Python Program To Represent Enum

In Python, you can use the enum module to create enumerations. Enumerations allow you to create symbolic names (members) for discrete values, making your code more readable and maintainable. Here's an example:

from enum import Enum, auto

class Color(Enum):
    RED = auto()
    GREEN = auto()
    BLUE = auto()

# Accessing enum members
print(Color.RED)   # Color.RED
print(Color.GREEN) # Color.GREEN
print(Color.BLUE)  # Color.BLUE

# Iterating over enum members
for color in Color:
    print(color)

# Comparison
if Color.RED == Color.RED:
    print("Color.RED is equal to Color.RED")

# Enum with custom values
class Status(Enum):
    SUCCESS = "Success"
    FAILURE = "Failure"

# Accessing enum members with custom values
print(Status.SUCCESS.value)  # "Success"
print(Status.FAILURE.value)  # "Failure"

In this example, we define two enumerations (Color and Status). The auto() function is used to automatically assign unique values to each member of the enumeration.

Note that members of an enumeration are distinct objects, and you can compare them using the standard equality operators (==, !=). Enum members can also have associated values, as shown in the Status enumeration.

Feel free to customize the enumerations according to your specific needs. Enumerations provide a way to represent a set of named values in a clean and organized manner.
