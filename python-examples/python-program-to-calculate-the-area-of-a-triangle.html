Python Program To Calculate The Area Of A Triangle

The area of a triangle can be calculated using the formula:

▲ Area = (base × height) ÷ 2

Here's a simple Python program that takes the base and height of a triangle as input, calculates its area, and prints the result:

# Get input from the user
base = float(input("Enter the base of the triangle: "))
height = float(input("Enter the height of the triangle: "))

# Calculate the area
area = (base * height) / 2

# Display the result
print("The area of the triangle with base", base, "and height", height, "is:", area)

In this program:

  1. input() is used to get input from the user. The float() function is used to convert the input to floating-point numbers.

  2. The area is calculated using the formula (base x height) ÷ 2.

  3. The result is then printed to the console using the print() function.

You can run this program, and it will prompt you to enter the base and height of the triangle. After entering the values, it will display the calculated area.

Example output:

Enter the base of the triangle: 8
Enter the height of the triangle: 5
The area of the triangle with base 8.0 and height 5.0 is: 20.0
