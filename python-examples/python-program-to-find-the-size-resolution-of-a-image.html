Python Program To Find The Size (Resolution) Of A Image

You can use the Python Imaging Library (PIL) or its fork, Pillow, to find the size (resolution) of an image.

You can install Pillow using the following command if you haven't installed it yet:

pip install Pillow

Note: The Pillow library is a fork of the original PIL library and is widely used for working with images in Python.

Below is an example using the Pillow library:

from PIL import Image

def get_image_size(image_path):
    try:
        # Open the image file
        with Image.open(image_path) as img:
            # Get the size (resolution) of the image
            width, height = img.size
            return width, height
    except Exception as e:
        print(f"Error: {e}")
        return None

# Example usage:
image_path = "path/to/your/image.jpg"  # Replace with the path to your image file
image_size = get_image_size(image_path)

if image_size:
    print(f"Image size (resolution): {image_size[0]} x {image_size[1]} pixels")

Make sure to replace "path/to/your/image.jpg" with the actual path to your image file. The get_image_size function attempts to open the image file using Pillow, retrieves the size (width and height) of the image, and prints the result.
