Python Program To Get The Last Element Of The List

You can get the last element of a list in Python using negative indexing or the [-1] index. Here's an example:

def get_last_element(my_list):
    if my_list:
        return my_list[-1]
    else:
        return None

# Example usage:
my_list = [1, 2, 3, 4, 5]

last_element = get_last_element(my_list)

if last_element is not None:
    print("Original List:", my_list)
    print("Last Element:", last_element)
else:
    print("The list is empty.")

Output:

Original List: [1, 2, 3, 4, 5]
Last Element: 5

In this example, the get_last_element function checks if the list is not empty before attempting to access the last element using negative indexing (my_list[-1]). If the list is empty, it returns None.

Make sure to handle the case where the list is empty to avoid potential errors when trying to access the last element.
