Python Program To Check The File Size

You can use the os.path.getsize() method of the os.path module in Python to check the size of a file. Here's an example:

import os

def get_file_size(file_path):
    try:
        # Get the size of the file in bytes
        file_size = os.path.getsize(file_path)
        return file_size
    except FileNotFoundError:
        print(f"Error: File '{file_path}' not found.")
        return None
    except Exception as e:
        print(f"Error: {e}")
        return None

# Example usage:
file_path = "path/to/your/file.txt"  # Replace with the path to your file

file_size = get_file_size(file_path)

if file_size is not None:
    print(f"File: {file_path}")
    print(f"File Size: {file_size} bytes")

Replace "path/to/your/file.txt" with the actual path to your file. The get_file_size function uses os.path.getsize to get the size of the file in bytes.

Adjust the program based on your specific requirements. The file_size variable will contain the size of the specified file.
