Python Program To Display Calendar

To display a calendar in Python, you can use the calendar module. Here's a simple program that prompts the user to enter the year and month, and then displays the calendar for that month:

import calendar

# Get input from the user
year = int(input("Enter the year (e.g., 2023): "))
month = int(input("Enter the month (1-12): "))

# Display the calendar
cal = calendar.month(year, month)
print(f"\nCalendar for {calendar.month_name[month]} {year}:\n")
print(cal)

Example:

Enter the month (1-12): 5

In this program, the calendar.month function is used to generate a string representation of the calendar for the specified year and month. The user is prompted to enter the year and month, and the program then displays the corresponding calendar.
