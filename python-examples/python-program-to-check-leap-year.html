Python Program To Check Leap Year

A leap year is a year that is evenly divisible by 4, except for years that are divisible by 100. However, years divisible by 400 are leap years. Here's a Python program to check if a given year is a leap year:

# Get input from the user
year = int(input("Enter a year: "))

# Check if the year is a leap year
if (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0):
    print(f"{year} is a leap year.")
else:
    print(f"{year} is not a leap year.")

In this program:

  1. input() is used to get input from the user. The int() function is used to convert the input to an integer.

  2. The program uses an if-else statement to check the leap year conditions. It checks if the year is divisible by 4 and not divisible by 100, or if it's divisible by 400.

  3. The result is then printed to the console.

You can run this program, and it will prompt you to enter a year. After entering the value, it will display whether the year is a leap year or not.

Example output:

Enter a year: 2020
2020 is a leap year.

Enter a year: 2021
2021 is not a leap year.
