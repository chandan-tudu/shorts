Python Program To Get A Substring Of A String

You can get a substring of a string in Python using slicing. Here's an example:

def get_substring(original_string, start_index, end_index):
    if start_index < 0 or end_index > len(original_string):
        print("Error: Invalid start or end index.")
        return None
    else:
        substring = original_string[start_index:end_index]
        return substring

# Example usage:
input_string = "Hello, World!"

# Get substring starting from index 7 to 12 (exclusive)
substring_result = get_substring(input_string, 7, 12)

if substring_result is not None:
    print("Original String:", input_string)
    print("Substring:", substring_result)

Output:

Original String: Hello, World!
Substring: World

In this example, the get_substring function takes the original string and the start and end indices as parameters and uses slicing (original_string[start_index:end_index]) to extract the substring. The function also checks if the provided indices are valid to avoid errors.

Adjust the start_index and end_index variables in the example to get the desired substring from your original string.
