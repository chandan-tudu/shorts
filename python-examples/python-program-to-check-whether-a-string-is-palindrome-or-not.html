Python Program To Check Whether A String Is Palindrome Or Not

Here's a Python program to check whether a given string is a palindrome or not:

def is_palindrome(input_str):
    # Convert the string to lowercase and remove spaces
    clean_str = input_str.lower().replace(" ", "")

    # Compare the original string with its reverse
    return clean_str == clean_str[::-1]

# Get input from the user
user_input = input("Enter a string: ")

# Check and display whether the string is a palindrome
if is_palindrome(user_input):
    print(f"{user_input} is a palindrome.")
else:
    print(f"{user_input} is not a palindrome.")

Example:

Enter a string: A man a plan a canal Panama
A man a plan a canal Panama is a palindrome.

In this program, the is_palindrome function takes a string as input, converts it to lowercase, and removes spaces. It then checks whether the cleaned string is equal to its reverse, and if so, the original string is a palindrome. The user is prompted to enter a string, and the program then checks and displays whether the entered string is a palindrome or not.
