Python Program To Count The Number Of Each Vowel

You can create a Python program to count the number of each vowel in a given string. Here's an example:

def count_vowels(input_string):
    # Convert the input string to lowercase to make the counting case-insensitive
    input_string = input_string.lower()

    # Define a set of vowels
    vowels = set("aeiou")

    # Initialize a dictionary to store the count of each vowel
    vowel_count = {vowel: 0 for vowel in vowels}

    # Iterate through each character in the input string
    for char in input_string:
        # Check if the character is a vowel
        if char in vowels:
            # Increment the count for the vowel in the dictionary
            vowel_count[char] += 1

    return vowel_count

# Example usage:
input_string = "Python Program to Count the Number of Each Vowel"
result = count_vowels(input_string)

print("Input string:", input_string)
print("Vowel counts:", result)

Output:

Input string: Python Program to Count the Number of Each Vowel
Vowel counts: {'e': 7, 'o': 7, 'a': 2, 'u': 2, 'i': 2}

In this program, the count_vowels function takes an input string, converts it to lowercase, initializes a dictionary to store the count of each vowel, and then iterates through each character in the string. If the character is a vowel, it increments the count in the dictionary. The example usage demonstrates how to use this function with a sample input string, and the output shows the count of each vowel in that string.
