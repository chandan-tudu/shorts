Python Program To Flatten A Nested List

You can flatten a nested list in Python using recursion or using built-in functions like itertools.chain and a list comprehension. Here's an example using a recursive approach:

def flatten_nested_list(nested_list):
    flattened_list = []
    for element in nested_list:
        if isinstance(element, list):
            flattened_list.extend(flatten_nested_list(element))
        else:
            flattened_list.append(element)
    return flattened_list

# Example usage:
nested_list = [1, [2, [3, 4], 5], 6, [7, 8]]

flattened_list = flatten_nested_list(nested_list)
print("Original Nested List:", nested_list)
print("Flattened List:", flattened_list)

Output:

Original Nested List: [1, [2, [3, 4], 5], 6, [7, 8]]
Flattened List: [1, 2, 3, 4, 5, 6, 7, 8]

This program defines a function flatten_nested_list that takes a nested list as input and recursively flattens it. The isinstance function is used to check if an element is a list, and extend is used to add the elements to the flattened list.

Alternatively, you can use the itertools.chain method along with a list comprehension:

from itertools import chain

nested_list = [1, [2, [3, 4], 5], 6, [7, 8]]

flattened_list = list(chain.from_iterable([item] if not isinstance(item, list) else item for item in nested_list))
print("Original Nested List:", nested_list)
print("Flattened List:", flattened_list)

This method creates an iterable for each element in the nested list and then flattens it using chain.from_iterable. The final result is converted to a list.
