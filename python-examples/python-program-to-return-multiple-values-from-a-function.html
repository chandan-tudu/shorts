Python Program To Return Multiple Values From A Function

In Python, you can return multiple values from a function using a tuple, a list, or any other iterable. Here's an example using a tuple:

def get_coordinates():
    x = 10
    y = 20
    z = 30
    return x, y, z

# Example usage:
result = get_coordinates()

# Unpack the tuple
x_coordinate, y_coordinate, z_coordinate = result

print(f"X Coordinate: {x_coordinate}")
print(f"Y Coordinate: {y_coordinate}")
print(f"Z Coordinate: {z_coordinate}")

Output:

X Coordinate: 10
Y Coordinate: 20
Z Coordinate: 30

In this example, the get_coordinates function returns a tuple containing three values (x, y, and z). When calling the function, the returned tuple is unpacked into individual variables (x_coordinate, y_coordinate, and z_coordinate).

You can also use other iterable types, such as lists or strings, to return multiple values:

def get_info():
    name = "John"
    age = 25
    city = "New York"
    return [name, age, city]

# Example usage:
result = get_info()

# Unpack the list
name, age, city = result

print(f"Name: {name}")
print(f"Age: {age}")
print(f"City: {city}")

Output:

Name: John
Age: 25
City: New York

Choose the approach that fits your needs, whether it's tuples, lists, or another iterable.
