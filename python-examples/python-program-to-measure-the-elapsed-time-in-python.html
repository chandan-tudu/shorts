Python Program To Measure The Elapsed Time In Python

You can measure the elapsed time in Python using the time module. Here's an example using the time.time() function:

import time

# Record the start time
start_time = time.time()

# Your code or task to measure the time for
for _ in range(1000000):
    pass

# Record the end time
end_time = time.time()

# Calculate the elapsed time
elapsed_time = end_time - start_time

print(f"Elapsed Time: {elapsed_time:.6f} seconds")

Output:

Elapsed Time: 0.034455 seconds

In this example, the time.time() function is used to record the current time before and after the code or task you want to measure. The difference between the start time and end time gives you the elapsed time.

Adjust the code inside the loop to represent the task or code snippet you want to measure. The elapsed time will be displayed in seconds. The :.6f in the formatted string is used to display the time with six decimal places.
