Python Program To Make A Simple Calculator

Here's a simple Python program that acts as a basic calculator. It can perform addition, subtraction, multiplication, and division:

def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    if y != 0:
        return x / y
    else:
        return "Cannot divide by zero"

# Get input from the user
num1 = float(input("Enter the first number: "))
operator = input("Enter an operator (+, -, *, /): ")
num2 = float(input("Enter the second number: "))

# Perform the calculation based on the operator
if operator == "+":
    result = add(num1, num2)
elif operator == "-":
    result = subtract(num1, num2)
elif operator == "*":
    result = multiply(num1, num2)
elif operator == "/":
    result = divide(num1, num2)
else:
    result = "Invalid operator"

# Display the result
print(f"The result of {num1} {operator} {num2} is: {result}")

Example:

Enter the first number: 10
Enter an operator (+, -, *, /): *
Enter the second number: 5
The result of 10 * 5 is: 50.0

In this program, the user is prompted to enter two numbers and an operator. Based on the entered operator, the program calls the corresponding function (add, subtract, multiply, or divide) to perform the calculation and then displays the result.
