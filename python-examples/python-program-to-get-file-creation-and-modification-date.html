Python Program To Get File Creation And Modification Date

You can use the os.path and datetime modules in Python to get the creation and modification dates of a file. Here's an example:

import os
from datetime import datetime

def get_file_dates(file_path):
    try:
        # Get file creation and modification times
        creation_time = os.path.getctime(file_path)
        modification_time = os.path.getmtime(file_path)

        # Convert timestamps to datetime objects
        creation_date = datetime.fromtimestamp(creation_time)
        modification_date = datetime.fromtimestamp(modification_time)

        return creation_date, modification_date
    except FileNotFoundError:
        print(f"Error: File '{file_path}' not found.")
        return None, None
    except Exception as e:
        print(f"Error: {e}")
        return None, None

# Example usage:
file_path = "path/to/your/file.txt"  # Replace with the path to your file

creation_date, modification_date = get_file_dates(file_path)

if creation_date is not None and modification_date is not None:
    print(f"File: {file_path}")
    print(f"Creation Date: {creation_date}")
    print(f"Modification Date: {modification_date}")

Replace "path/to/your/file.txt" with the actual path to your file. The get_file_dates function uses os.path.getctime and os.path.getmtime to get the creation and modification times of the file, respectively. The timestamps are then converted to datetime objects for easier handling.

Adjust the program based on your specific requirements. The creation_date and modification_date variables will contain the creation and modification dates of the specified file.
