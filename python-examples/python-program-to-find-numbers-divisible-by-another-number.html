Python Program To Find Numbers Divisible By Another Number

Here's a Python program that finds numbers divisible by another number within a specified range:

def find_divisible_numbers(start, end, divisor):
    divisible_numbers = [num for num in range(start, end + 1) if num % divisor == 0]
    return divisible_numbers

# Get input from the user
start_range = int(input("Enter the start of the range: "))
end_range = int(input("Enter the end of the range: "))
divisor = int(input("Enter the divisor: "))

# Find and display numbers divisible by the given divisor in the specified range
result = find_divisible_numbers(start_range, end_range, divisor)
print(f"Numbers divisible by {divisor} in the range [{start_range}, {end_range}]: {result}")

Example:

Enter the start of the range: 1
Enter the end of the range: 20
Enter the divisor: 3
Numbers divisible by 3 in the range [1, 20]: [3, 6, 9, 12, 15, 18]

In this program, the find_divisible_numbers function takes the start and end of the range, along with the divisor, and uses a list comprehension to find numbers within the range that are divisible by the given divisor. The user is prompted to enter the start and end of the range, as well as the divisor, and the program then finds and displays the numbers that satisfy the condition.
