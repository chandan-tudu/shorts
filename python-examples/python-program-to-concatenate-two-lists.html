Python Program To Concatenate Two Lists

You can concatenate two lists in Python using the + operator or the extend() method. Here are examples of both approaches:

1. Using the + Operator:

def concatenate_lists(list1, list2):
    concatenated_list = list1 + list2
    return concatenated_list

# Example usage:
list_a = [1, 2, 3]
list_b = ['a', 'b', 'c']

result = concatenate_lists(list_a, list_b)
print("List A:", list_a)
print("List B:", list_b)
print("Concatenated List:", result)

Output:

List A: [1, 2, 3]
List B: ['a', 'b', 'c']
Concatenated List: [1, 2, 3, 'a', 'b', 'c']

2. Using the extend() Method:

def concatenate_lists(list1, list2):
    list1.extend(list2)
    return list1

# Example usage:
list_a = [1, 2, 3]
list_b = ['a', 'b', 'c']

result = concatenate_lists(list_a, list_b)
print("List A:", list_a)
print("List B:", list_b)
print("Concatenated List:", result)

Output:

List A: [1, 2, 3, 'a', 'b', 'c']
List B: ['a', 'b', 'c']
Concatenated List: [1, 2, 3, 'a', 'b', 'c']

Both approaches achieve the same result. Choose the one that fits your coding style and requirements. Note that using the extend() method modifies the original list, while the + operator creates a new list.
