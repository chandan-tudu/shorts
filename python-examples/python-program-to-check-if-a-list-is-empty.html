Python Program To Check If A List Is Empty

You can check if a list is empty in Python using a simple if statement. Here's an example:

def is_list_empty(my_list):
    if not my_list:
        return True
    else:
        return False

# Example usage:
empty_list = []
non_empty_list = [1, 2, 3]

print("Is empty_list empty?", is_list_empty(empty_list))
print("Is non_empty_list empty?", is_list_empty(non_empty_list))

Output:

Is empty_list empty? True
Is non_empty_list empty? False

In this example, the is_list_empty function takes a list as an argument and returns True if the list is empty (evaluates to False in a boolean context) and False otherwise.

Alternatively, you can directly use the list in an if statement:

empty_list = []
non_empty_list = [1, 2, 3]

if not empty_list:
    print("empty_list is empty")

if not non_empty_list:
    print("non_empty_list is empty")
else:
    print("non_empty_list is not empty")

This approach is more concise and idiomatic in Python. The if not my_list checks if the list is empty.
