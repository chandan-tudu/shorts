Python Program To Sort Words In Alphabetic Order

You can create a Python program to sort words in alphabetical order using the sorted() function. Here's an example:

def sort_words_alphabetically(sentence):
    # Split the sentence into a list of words
    words = sentence.split()

    # Sort the words in alphabetical order
    sorted_words = sorted(words)

    # Join the sorted words into a new sentence
    sorted_sentence = ' '.join(sorted_words)

    return sorted_sentence

# Example usage:
input_sentence = "Python is a powerful programming language"
result = sort_words_alphabetically(input_sentence)

print("Original sentence:", input_sentence)
print("Sorted sentence:", result)

Output:

Original sentence: Python is a powerful programming language
Sorted sentence: Python a is language powerful programming

This program defines a function sort_words_alphabetically that takes a sentence as input, splits it into words, sorts the words alphabetically, and then joins them back into a sentence. The example usage demonstrates how to use this function with a sample sentence.
