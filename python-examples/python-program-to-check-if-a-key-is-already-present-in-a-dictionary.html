Python Program To Check If A Key Is Already Present In A Dictionary

You can check if a key is already present in a dictionary in Python using the in keyword or the get() method. Here are examples of both approaches:

1. Using the in Keyword:

def check_key_in_dict(dictionary, key):
    if key in dictionary:
        print(f"The key '{key}' is present in the dictionary.")
    else:
        print(f"The key '{key}' is not present in the dictionary.")

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}

check_key_in_dict(my_dict, 'b')
check_key_in_dict(my_dict, 'x')

Output:

The key 'b' is present in the dictionary.
The key 'x' is not present in the dictionary.

2. Using the get() Method:

def check_key_in_dict(dictionary, key):
    value = dictionary.get(key)
    if value is not None:
        print(f"The key '{key}' is present in the dictionary with value: {value}")
    else:
        print(f"The key '{key}' is not present in the dictionary.")

# Example usage:
my_dict = {'a': 1, 'b': 2, 'c': 3}

check_key_in_dict(my_dict, 'b')
check_key_in_dict(my_dict, 'x')

Output:

The key 'b' is present in the dictionary with value: 2
The key 'x' is not present in the dictionary.

Choose the method that best fits your requirements. The in keyword is straightforward for a simple check, while the get() method also allows you to retrieve the value associated with the key if it exists.
