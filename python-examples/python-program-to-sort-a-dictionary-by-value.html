Python Program To Sort A Dictionary By Value

To sort a dictionary by its values in Python, you can use the sorted() function along with a lambda function as the key argument. Here's an example:

# Sample dictionary
my_dict = {'apple': 3, 'banana': 1, 'orange': 2, 'grape': 4}

# Sort the dictionary by value
sorted_dict = dict(sorted(my_dict.items(), key=lambda item: item[1]))

# Print the sorted dictionary
print("Original Dictionary:", my_dict)
print("Sorted Dictionary by Value:", sorted_dict)

Output:

Original Dictionary: {'apple': 3, 'banana': 1, 'orange': 2, 'grape': 4}
Sorted Dictionary by Value: {'banana': 1, 'orange': 2, 'apple': 3, 'grape': 4}

In this example, the items() method is used to get a list of key-value pairs from the dictionary, and the sorted() function is applied with a lambda function as the key to sort the items based on their values (item[1]). The result is then converted back to a dictionary.

If you want to sort the dictionary in descending order, you can use the reverse=True argument in the sorted() function:

# Sort the dictionary by value in descending order
sorted_dict_desc = dict(sorted(my_dict.items(), key=lambda item: item[1], reverse=True))

# Print the sorted dictionary in descending order
print("Sorted Dictionary by Value (Descending):", sorted_dict_desc)

Output:

Sorted Dictionary by Value (Descending): {'grape': 4, 'apple': 3, 'orange': 2, 'banana': 1}

Adjust the code as needed for your specific use case.
