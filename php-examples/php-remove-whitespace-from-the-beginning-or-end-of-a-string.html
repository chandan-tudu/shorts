PHP - Remove Whitespace From The Beginning Or End Of A String

To remove whitespace from the beginning or end of a string in PHP, you can use the trim() function. Here's an example:

<?php
$string = "   Hello, World!   ";
$trimmedString = trim($string);

echo $trimmedString;
?>

In this example, the trim() function is applied to the $string variable, removing any leading and trailing whitespace. The output will be:

Hello, World!

If you only want to remove whitespace from the beginning (left) or the end (right) of the string, you can use ltrim() or rtrim() respectively:

<?php
$string = "   Hello, World!   ";

$leftTrimmedString = ltrim($string);
$rightTrimmedString = rtrim($string);

echo "Left Trimmed: $leftTrimmedString\n";
echo "Right Trimmed: $rightTrimmedString\n";
?>

This will output:

Left Trimmed: Hello, World!
Right Trimmed:    Hello, World!

Choose the appropriate function (trim(), ltrim(), or rtrim()) based on your specific needs.
