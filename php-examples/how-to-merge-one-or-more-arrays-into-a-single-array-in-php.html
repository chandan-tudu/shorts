How To Merge One Or More Arrays Into A Single Array In PHP

In PHP, you can merge one or more arrays into a single array using the array_merge() function. The array_merge() function takes two or more arrays as arguments and returns a new array containing all the values of the input arrays. Here's an example:

<?php
$array1 = array('a', 'b', 'c');
$array2 = array('d', 'e', 'f');
$array3 = array('g', 'h', 'i');

$mergedArray = array_merge($array1, $array2, $array3);

print_r($mergedArray);
?>

Output:

Array
(
    [0] => a
    [1] => b
    [2] => c
    [3] => d
    [4] => e
    [5] => f
    [6] => g
    [7] => h
    [8] => i
)

In this example, array_merge($array1, $array2, $array3) merges the three arrays into a single array.

If the arrays you want to merge have string keys, and you want to preserve those keys, you can use the + operator or the array_merge() function with associative arrays:

<?php
$array1 = array('a' => 1, 'b' => 2, 'c' => 3);
$array2 = array('b' => 4, 'c' => 5, 'd' => 6);

// Using the + operator
$mergedArray = $array1 + $array2;

// Using array_merge with associative arrays
$mergedArrayAlt = array_merge($array1, $array2);

print_r($mergedArray);
print_r($mergedArrayAlt);
?>

Output:

Array
(
    [a] => 1
    [b] => 2
    [c] => 3
    [d] => 6
)
Array
(
    [a] => 1
    [b] => 2
    [c] => 3
    [d] => 6
)

Both methods preserve the keys from the original arrays. The + operator gives precedence to the elements in the leftmost array when there are duplicate keys. The array_merge() function, on the other hand, takes the values from the last duplicate key encountered.
