PHP - Counts The Number Of Times A Substring Occurs In A String

To count the number of times a substring occurs in a string in PHP, you can use the substr_count() function. Here's an example:

<?php
$string = "apple banana apple orange apple";
$substring = "apple";

$count = substr_count($string, $substring);

echo "The substring '$substring' occurs $count times in the string.";
?>

In this example, the substr_count() function is used to count the occurrences of the substring "apple" in the given string. The output will be:

The substring 'apple' occurs 3 times in the string.

The substr_count() function takes two parameters:

  1. The original string in which you want to search for the substring.
  2. The substring you want to count.

It returns the number of occurrences of the substring in the original string.
