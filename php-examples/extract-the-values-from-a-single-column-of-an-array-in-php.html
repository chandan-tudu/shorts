Extract The Values From A Single Column Of An Array In PHP

To extract the values from a single column of an array in PHP, you can use the array_column() function.

This function returns the values from a single column of an array identified by the specified column key or index. Here's an example:

<?php
// Original multidimensional array
$data = array(
    array('id' => 1, 'name' => 'John', 'age' => 25),
    array('id' => 2, 'name' => 'Jane', 'age' => 30),
    array('id' => 3, 'name' => 'Doe', 'age' => 22)
);

// Extract values from the 'name' column
$names = array_column($data, 'name');

print_r($names);
?>

Output:

Array
(
    [0] => John
    [1] => Jane
    [2] => Doe
)

In this example, array_column($data, 'name') extracts the values from the 'name' column of the original array.

You can adjust the column key or index as needed based on your array structure. If the array is associative and you want to extract values based on numeric indices, you can use the numeric index instead of a column key.
