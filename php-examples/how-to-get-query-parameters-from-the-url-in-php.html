How To Get Query Parameters From The URL In PHP

In PHP, you can retrieve query parameters from the URL using the $_GET superglobal array. The $_GET array contains key-value pairs of query parameters passed in the URL. Here's a simple example:

<?php
// Assuming the URL is http://example.com/page.php?name=John&age=25

// Retrieve individual query parameters
$name = $_GET['name'];
$age = $_GET['age'];

// Output the values
echo "Name: " . $name . "<br>";
echo "Age: " . $age . "<br>";
?>

In the example above, if the URL is http://example.com/page.php?name=John&age=25, the output will be:

Name: John
Age: 25

Keep in mind that you should validate and sanitize user input, especially when dealing with data from URLs, to prevent security issues like SQL injection or cross-site scripting (XSS) attacks. Additionally, you may want to check if the parameters exist before accessing them to avoid errors:

<?php
// Assuming the URL is http://example.com/page.php?name=John&age=25

// Check if the 'name' parameter is set
if (isset($_GET['name'])) {
    $name = $_GET['name'];
    echo "Name: " . $name . "<br>";
} else {
    echo "Name parameter not set.<br>";
}

// Check if the 'age' parameter is set
if (isset($_GET['age'])) {
    $age = $_GET['age'];
    echo "Age: " . $age . "<br>";
} else {
    echo "Age parameter not set.<br>";
}
?>

This helps prevent errors when trying to access non-existent query parameters.
