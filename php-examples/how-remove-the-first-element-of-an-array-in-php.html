How Remove The First Element Of An Array In PHP

To remove the first element of an array in PHP, you can use the array_shift() function. Here's an example:

<?php
$fruits = array('apple', 'banana', 'cherry', 'date');

// Remove the first element
array_shift($fruits);

print_r($fruits);
?>

This would output:

Array
(
    [0] => banana
    [1] => cherry
    [2] => date
)

The array_shift() function removes the first element from the array and returns its value. Keep in mind that this operation modifies the original array.

If you want to preserve the original array and create a new array without the first element, you can use the array_slice() function:

<?php
$fruits = array('apple', 'banana', 'cherry', 'date');

// Create a new array without the first element
$newArray = array_slice($fruits, 1);

print_r($newArray);

This would output:

Array
(
    [0] => banana
    [1] => cherry
    [2] => date
)

Both array_shift() and array_slice() provide different ways to achieve the removal of the first element based on whether you want to modify the original array or create a new array without the first element. Choose the method that suits your specific use case.
