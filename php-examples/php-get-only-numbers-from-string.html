PHP Get Only Numbers From String

In PHP, you can extract only the numbers from a string using regular expressions and the preg_replace() function. Here's an example:

<?php
$string = "The price is $25.99 for 3 items.";

// Use regular expression to extract only numbers
$numbersOnly = preg_replace('/[^0-9]/', '', $string);

// Display the result
echo "Numbers only: $numbersOnly";
?>

Output:

Numbers only: 25993

In this example, the regular expression /[^0-9]/ is used with preg_replace() to remove any characters that are not numbers (0-9) from the original string. The resulting string contains only the numbers.

If you also want to extract decimal numbers, you can modify the regular expression to /[^0-9.]/:

$numbersOnly = preg_replace('/[^0-9.]/', '', $string);

This will allow the inclusion of the decimal point (.) in the extracted numbers.

Note that if the string contains multiple numbers, the extracted result will be a concatenated string of all the numbers.

If you need to work with individual numbers, you may consider using the preg_match_all() function to capture all occurrences into an array:

$string = "The price is $25.99 for 3 items. Total: $50.50";

// Use regular expression to extract all numbers
preg_match_all('/[0-9]+(?:\.[0-9]+)?/', $string, $matches);

// Display the result
print_r($matches[0]);

Output:

Array
(
    [0] => 25.99
    [1] => 3
    [2] => 50.50
)

In this example, $matches[0] is an array containing all the extracted numbers.
