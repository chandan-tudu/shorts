PHP String Lowercase And Uppercase

In PHP, you can convert a string to lowercase using strtolower() and to uppercase using strtoupper(). Here's an example that demonstrates both:

<?php
// Original string
$originalString = "Hello, World!";

// Convert the string to lowercase
$lowercaseString = strtolower($originalString);

// Convert the string to uppercase
$uppercaseString = strtoupper($originalString);

// Display the results
echo "Original String: $originalString<br>";
echo "Lowercase String: $lowercaseString<br>";
echo "Uppercase String: $uppercaseString";
?>

Output:

Original String: Hello, World!
Lowercase String: hello, world!
Uppercase String: HELLO, WORLD!

In this example:

  • $lowercaseString = strtolower($originalString); converts all alphabetic characters in $originalString to lowercase.
  • $uppercaseString = strtoupper($originalString); converts all alphabetic characters in $originalString to uppercase.

These functions are handy when you need to standardize the case of a string for comparison or display purposes.
