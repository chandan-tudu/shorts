How To Sort Arrays In PHP

In PHP, you can sort an array using various functions based on your sorting requirements. Here are some commonly used sorting functions:

1. "sort()": Sort an array in ascending order.

<?php
$myArray = array(3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5);
sort($myArray);

// Display the sorted array
print_r($myArray);
?>

Output:

Array
(
    [0] => 1
    [1] => 1
    [2] => 2
    [3] => 3
    [4] => 3
    [5] => 4
    [6] => 5
    [7] => 5
    [8] => 5
    [9] => 6
    [10] => 9
)

2. "rsort()": Sort an array in descending order.

<?php
$myArray = array(3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5);
rsort($myArray);

// Display the reverse sorted array
print_r($myArray);
?>

Output:

Array
(
    [0] => 9
    [1] => 6
    [2] => 5
    [3] => 5
    [4] => 5
    [5] => 4
    [6] => 3
    [7] => 3
    [8] => 2
    [9] => 1
    [10] => 1
)

3. "asort()": Sort an associative array in ascending order based on values.

<?php
$myArray = array('apple' => 3, 'banana' => 1, 'orange' => 4);
asort($myArray);

// Display the sorted associative array
print_r($myArray);
?>

Output:

Array
(
    [banana] => 1
    [apple] => 3
    [orange] => 4
)

4. "arsort()": Sort an associative array in descending order based on values.

<?php
$myArray = array('apple' => 3, 'banana' => 1, 'orange' => 4);
arsort($myArray);

// Display the reverse sorted associative array
print_r($myArray);
?>

Output:

Array
(
    [orange] => 4
    [apple] => 3
    [banana] => 1
)

5. "ksort()": Sort an associative array in ascending order based on keys.

<?php
$myArray = array('apple' => 3, 'banana' => 1, 'orange' => 4);
ksort($myArray);

// Display the sorted associative array based on keys
print_r($myArray);
?>

Output:

Array
(
    [apple] => 3
    [banana] => 1
    [orange] => 4
)

6. "krsort()": Sort an associative array in descending order based on keys.

<?php
$myArray = array('apple' => 3, 'banana' => 1, 'orange' => 4);
krsort($myArray);

// Display the reverse sorted associative array based on keys
print_r($myArray);
?>

Output:

Array
(
    [orange] => 4
    [banana] => 1
    [apple] => 3
)

These functions sort the arrays in place, meaning they modify the original array. If you want to preserve the original array and create a sorted version, you can assign the sorted result to a new variable.
