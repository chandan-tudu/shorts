PHP Insert An Element Into An Array At A Specific Index

In PHP, you can insert an element into an array at a specific index using the array_splice() function. Here's an example:

<?php
// Original array
$originalArray = array("apple", "banana", "orange");

// Element to insert
$newElement = "grape";

// Index at which to insert the new element
$indexToInsert = 1;

// Insert the new element into the array at the specified index
array_splice($originalArray, $indexToInsert, 0, $newElement);

// Display the updated array
print_r($originalArray);
?>

In this example:

  • $originalArray is the original array.
  • $newElement is the element you want to insert.
  • $indexToInsert is the index at which you want to insert the new element.

The array_splice() function modifies the original array by inserting the new element at the specified index. The third parameter (0) specifies the number of elements to be removed at the specified index (in this case, none).

Output:

Array
(
    [0] => apple
    [1] => grape
    [2] => banana
    [3] => orange
)

As a result, "grape" is inserted at index 1, and the elements after that are shifted to accommodate the new element.

Note that array_splice() modifies the original array in place. If you want to keep the original array unchanged, you can create a copy of it before using array_splice():

$originalArray = array("apple", "banana", "orange");
$copyArray = $originalArray;

array_splice($copyArray, $indexToInsert, 0, $newElement);
