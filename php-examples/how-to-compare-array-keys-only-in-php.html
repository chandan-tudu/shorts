How To Compare Array Keys Only In PHP

To compare array keys only in PHP, you can use various functions such as array_keys(), array_diff(), or array_intersect(). Here are some examples:

1. Using "array_keys()" and "array_diff()":

<?php
$array1 = array('a' => 1, 'b' => 2, 'c' => 3);
$array2 = array('a' => 1, 'b' => 2, 'c' => 3, 'd' => 4);

$keys1 = array_keys($array1);
$keys2 = array_keys($array2);

$diff = array_diff($keys1, $keys2);

if (empty($diff)) {
    echo "Arrays have the same keys.";
} else {
    echo "Arrays have different keys.";
}
?>

2. Using "array_keys()" and "array_intersect()":

<?php
$array1 = array('a' => 1, 'b' => 2, 'c' => 3);
$array2 = array('a' => 1, 'b' => 2, 'c' => 3, 'd' => 4);

$keys1 = array_keys($array1);
$keys2 = array_keys($array2);

$intersect = array_intersect($keys1, $keys2);

if (count($intersect) == count($keys1) && count($intersect) == count($keys2)) {
    echo "Arrays have the same keys.";
} else {
    echo "Arrays have different keys.";
}
?>

In both examples, array_keys() is used to extract the keys from each array, and then either array_diff() or array_intersect() is used to compare the keys. The conditions check if the resulting array after the comparison has the same number of elements as the original arrays.

Choose the method that fits your specific use case and comparison criteria.
