PHP Regular Expression To Extract String

To extract a specific pattern or substring from a string in PHP, you can use regular expressions along with the preg_match() function. Here's a simple example:

<?php
// Original string
$string = "The code for GPT-3 is AI42GPT335.";

// Regular expression to extract the alphanumeric code
$pattern = '/GPT-\d+/';

// Perform the regular expression match
if (preg_match($pattern, $string, $matches)) {
    // Display the extracted code
    echo "Extracted code: " . $matches[0];
} else {
    echo "No match found!";
}
?>

Output:

Extracted code: GPT-3

In this example, the regular expression /GPT-\d+/ is used to match the pattern "GPT-" followed by one or more digits. The preg_match() function is then used to perform the match against the $string. If a match is found, the extracted code is displayed; otherwise, it prints "No match found!"

Adjust the regular expression according to your specific pattern or substring extraction requirements. Regular expressions provide a powerful and flexible way to search and manipulate strings based on patterns.
