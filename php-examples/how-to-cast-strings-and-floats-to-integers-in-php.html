How To Cast Strings And Floats To Integers In PHP

In PHP, you can cast strings and floats to integers using explicit type casting or type conversion functions. Here are examples for both scenarios:

1. Explicit Type Casting:

You can explicitly cast a variable to an integer using (int):

<?php
$numericString = "42";
$floatNumber = 3.14;

$integerFromString = (int)$numericString;
$integerFromFloat = (int)$floatNumber;

echo "Casting string to int: $integerFromString\n";
echo "Casting float to int: $integerFromFloat";
?>

Output:

Casting string to int: 42
Casting float to int: 3

2. Type Conversion Functions:

You can use type conversion functions such as intval() and floatval():

<?php
$numericString = "42";
$floatNumber = 3.14;

$integerFromString = intval($numericString);
$integerFromFloat = intval($floatNumber);

echo "Converting string to int: $integerFromString\n";
echo "Converting float to int: $integerFromFloat";
?>

Output:

Converting string to int: 42
Converting float to int: 3

Both approaches will convert the string or float to an integer. It's important to note that when casting a float to an integer, the decimal part is truncated (not rounded).

Choose the method that best fits your coding style and requirements. If the conversion is not possible (for example, if the string contains non-numeric characters), the result will be 0 when using type casting or intval(). If using (int), the result will be 0, and a notice will be issued if error reporting is enabled.
