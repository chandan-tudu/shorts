How To Remove Last Character Of A String In PHP

You can remove the last character of a string in PHP using various approaches. Here are a couple of examples:

1. Using "substr" Function:

The substr function can be used to extract a portion of a string. To remove the last character, you can extract the substring excluding the last character.

<?php
$str = "Hello World";
$newStr = substr($str, 0, -1);
echo $newStr; // Output: Hello Worl
?>

2. Using "substr_replace" Function:

The substr_replace function can be used to replace a part of a string with another string. In this case, you replace the last character with an empty string.

<?php
$str = "Hello World";
$newStr = substr_replace($str, "", -1);
echo $newStr; // Output: Hello Worl
?>

3. Using "rtrim" Function:

The rtrim function is typically used to remove whitespace from the end of a string, but you can also use it to remove a specific character if it occurs at the end.

<?php
$str = "Hello World";
$charToRemove = 'd';
$newStr = rtrim($str, $charToRemove);
echo $newStr; // Output: Hello Worl
?>

Choose the method that best fits your requirements and coding style.
