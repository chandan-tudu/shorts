How To Reverse An Array In PHP

In PHP, you can reverse an array using the array_reverse() function. Here's an example:

<?php
// Sample array
$myArray = array('value1', 'value2', 'value3');

// Reverse the array
$reversedArray = array_reverse($myArray);

// Display the reversed array
print_r($reversedArray);
?>

In this example, the array_reverse() function is used to reverse the order of elements in the $myArray. The reversed array is then stored in the variable $reversedArray, and print_r() is used to display the result. The output will be:

Array
(
    [0] => value3
    [1] => value2
    [2] => value1
)

Note that array_reverse() does not modify the original array; it returns a new array with the elements in reverse order. If you want to reverse the original array in-place, you can use the array_reverse() function with the optional second parameter set to true:

$myArray = array('value1', 'value2', 'value3');
array_reverse($myArray, true);
print_r($myArray);

However, in most cases, you would prefer to keep the original array unchanged and work with the reversed version as needed.
